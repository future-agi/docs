The Administration Panel centralizes everything you need to monitor usage, manage access, configure integrations, track billing, and maintain your personal settings.

## Usage Summary

Provides full visibility into how the platform is being consumed.

### Key Features

- **Overview Dashboard**
  - Total evaluations (by module: Resource Used, Evals, Protect, Prompt Optimization)
  - Credit balance and consumption
  - Real-time protection events (e.g. filtered or flagged content)

- **Date Filtering**
  - Pre-set ranges (Last 7 days, Last 30 days) or custom range
  - Zoom in/out on time-series charts

- **Drill-down Views**
  1. By feature: switch between Evaluations, Prompt Optimization, Protect
  2. By user or API key: filter consumption per sub-user

- **Export & Reporting**
  - Download CSV, XLSX, or PDF
  - Schedule recurring exports via email
![usage-summary](/images/usage-summary.png)
### Step-by-step Guide

1. Go to Settings > Usage Summary
2. Select your date range
3. Use module tabs to toggle features
4. Hover over charts for exact numbers
5. Click Export to download or schedule reports

## API Keys & User Management

### Managing Users

- **Invite new users**
  1. Go to Settings > User Management
  2. Click Add User
  3. Enter email and assign role (Admin, Editor, Viewer)
  4. Submit – they'll receive an invitation link

- **Modify or revoke access**
  - Use the menu next to a user to Edit Role, Resend Invite, or Remove User

- **View pending invitations** under the Invitations tab
![user-management](/images/user-management.png)
### Managing API Keys

- **Add a new key**
  1. Go to Settings > API Keys
  2. Click Add Key
  3. Choose provider (OpenAI, Azure, Anthropic, Hugging Face, etc.)
  4. Paste or upload key/JSON credentials
  5. Save

- **Encryption & Safety**
  - All keys are encrypted at rest and in transit
  - Only Admin users can view keys

- **Rotate or delete keys**
  - Rotate uploads a new key without downtime
  - Delete removes access permanently

## Billing Management

### Plan Management

- Redesigned with a cleaner look and simple navigation  
- Compare available tiers (Free, Growth, Enterprise) easily with Pricing Cards  
- Estimate costs using the Price Calculator  
- View a clear Plan Breakdown for features and benefits

### Track & Add Credits

- Current balance displayed at the top of the page
- Usage history table showing date, feature, and amount consumed
- **To add credits**
  1. Click Top Up
  2. Enter amount
  3. Choose payment method (Credit Card, ACH, Wire)
  4. Confirm
- **Invoices & payments**
  - Download past invoices (PDF)
  - Update or add payment methods

## Profile Management

### Reset Password

Keep your account secure by managing credentials.

1. Go to Settings > Profile
2. Under Security, click Reset Password
3. Enter current password, then new password twice
4. Save Changes
5. You'll receive a confirmation email once updated

<Note>We recommend strong passwords (at least 12 characters with letters, numbers, and symbols).</Note>
