This guide provides step-by-step instructions for navigating the **Administration Panel**, managing **API Keys**, configuring **Custom Models**, tracking **Usage Summary**, handling **Billing & Plans**, and updating **User Profile settings**.

---

## **ADMINISTRATION**

### **1. Usage Summary**

This section provides a detailed view of token consumption, evaluation usage, and system performance. Users can monitor their usage trends through graphical representations and filter data by selecting a specific date range. Additionally, reports can be exported for billing and resource tracking.

 

- Go to **Settings > Usage Summary**.
- Select a **date range** to filter data.
- Review the **graphical representation** of usage trends.

### **2. User Management**

This section allows administrators to manage team members, assign roles, and control access permissions. Users can view a list of registered members, invite new users, modify existing roles, and approve or reject pending invitations. Removing team members from the system can also be done from this section.

 

- Go to **Settings > User Management**.
- View the list of registered users.
- Click **Add User** to invite a new team member.
1. Enter the **user’s email address**.
2. Select an **access role** from the dropdown.
3. Click **Submit** to send the invitation.

### **3. API Keys**

This section enables users to configure API access for third-party integrations such as OpenAI, Azure, Anthropic, and Hugging Face. 

 

- Go to **Settings > API Keys**.
- View the list of existing API keys.
1. Select the **provider** (e.g., OpenAI, Azure, Anthropic).
2. Enter the **API Key** in the designated field.
3. Click **Save**.

### **4. Custom Models**

This section allows users to integrate third-party AI models from providers such as Vertex AI, AWS SageMaker, and AWS Bedrock. Users can register and manage their models by configuring authentication credentials and defining input and output token costs.

 

- Go to **Settings > Custom Models**.
- View the list of available custom models.
- Click **Add Model** to configure a new model.
- Select a **Model Provider** (e.g., Vertex AI, AWS SageMaker, AWS Bedrock).
- Enter the required details:
    - **Model Name**
    - **Input Token Cost**
    - **Output Token Cost**
    - **Provider-Specific Credentials** (e.g., API keys, JSON credentials, region settings).
- Click **Save**.

## **PLAN SUMMARY**

### **1. Plans & Pricing**

This section  enables users to select and manage their subscription plans. Users can compare different pricing tiers, toggle between monthly and yearly billing cycles, and upgrade or downgrade their plans based on their needs.

 

- Go to **Settings > Plans & Pricing**.
- View available subscription plans.
- Select the **subscription tier** that best suits your requirements.
