## Overview

Future AGI's Python client library helps you interact with the Future AGI platform to:

- Log and trace model events
- Run automated evaluations
- Validate model outputs
- Monitor model performance

## Prerequisites

Before getting started, you'll need:

- Python 3.6 or higher
- A Future AGI account with API credentials
- Basic understanding of LLM applications
