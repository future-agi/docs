### Evaluation Summary

- Added a summary section for each type of evaluation in the dataset  
- View data visually in graphs for different evaluation types  
- Compare evaluation results by toggling between the same type of evaluations  

### Prompt Summary

- Added a summary section for each run-prompt in the dataset  
- View data visually in bar graphs and compare results  
- Track average tokens used, average response time, and average cost for prompts  

### Annotation Summary

- Added a summary section for each annotation in the dataset  
- View data visually in graphs for numeric, text, and option types  
- Track time spent on each annotation by annotator  
- Access detailed insights for every annotation type graph
