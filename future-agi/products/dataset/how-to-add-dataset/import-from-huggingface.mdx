---
title: Importing from Hugging Face
description: Import a dataset from Hugging Face.
---


## 1. Adding Dataset
Select **Build** from top left corner under **Main** section. Then click on **Add Dataset**.
![Step 1](./images/import-from-huggingface/step1.png)

## 2. Selecting Hugging Face Option
Now choose **Import from Hugging Face** option to import dataset directly from Hugging Face.
![Step 2](./images/import-from-huggingface/step2.png)

## 3. Providing Hugging Face Dataset ID
Copy the dataset name from https://huggingface.co/datasets in which you want to perform experimentaion. Then paste the dataset name and click on **Load Dataset**.
![Step 3](./images/import-from-huggingface/step3.png)

## 4. Providing Additional Dataset Details
Give a **name** to this imported dataset. Then choose your **model type** from the drop down menu, e.g. Generative LLM. After that, choose **subset** and **split** of the dataset. (Read the dataset card from Hugging Face for more details). Then click on **Start Experimenting**.
![Step 4](./images/import-from-huggingface/step4.png)

## 5. Dataset is Ready for Experimention
You can now see the imported dataset on your dashboard. If not showing then refresh the page.
![Step 5](./images/import-from-huggingface/step5.png)
