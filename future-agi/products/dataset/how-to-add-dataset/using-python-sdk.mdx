## Installation

First, install the Future AGI Python client:

```bash
pip install futureagi

export FI_API_KEY="your_api_key"
export FI_SECRET_KEY="your_secret_key"
```

## Creating Datasets

### 1. Initialize the Client

```python
from fi.datasets import DatasetClient, Dataset
from fi.utils.types import ModelTypes

# Initialize with your API credentials
client = DatasetClient()
```

### 2. Create Empty Dataset

```python
# Configure dataset
dataset_config = Dataset(
    name="my_dataset",
    model_type=ModelTypes.GENERATIVE_LLM
)

# Create empty dataset
dataset = DatasetClient.create_empty_dataset(config=dataset_config)
```

### 3. Create Dataset from File

Supported file formats: CSV, Excel (.xlsx, .xls), JSON, and JSONL.

```python
# Create dataset from local file
dataset = DatasetClient.create_dataset_from_file(
    file_path="path/to/your/data.csv",
    dataset_config=Dataset(
        name="file_dataset",
        model_type=ModelTypes.GENERATIVE_LLM
    )
)
```

### 4. Create Dataset from Hugging Face

```python
from fi.datasets.types import HuggingfaceDatasetConfig

# Create dataset from Hugging Face
dataset = DatasetClient.create_dataset_from_huggingface(
    huggingface_dataset_path="dataset_name",
    dataset_config=Dataset(
        name="huggingface_dataset",
        model_type=ModelTypes.GENERATIVE_LLM
    ),
    huggingface_dataset_config=HuggingfaceDatasetConfig(
        split="train",  # Optional: specify dataset split
        num_rows=1000   # Optional: limit number of rows
    )
)
```

## Working with Datasets

### Download Dataset

```python
# Download as DataFrame or save to file
dataset_table = DatasetClient.download_dataset(
    dataset_name="my_dataset",
    file_path="output.csv"  # Optional: save to file
)
```

### Delete Dataset

```python
# Delete a dataset
DatasetClient.delete_dataset(dataset_name="my_dataset")
```

## Data Types

When working with datasets, the SDK automatically detects and supports multiple data types for columns by analyzing the provided data.

- Text
- Boolean
- Integer
- Float
- JSON
- Array
- Image
- Datetime
