title: Adding from Existing Dataset
description: Choose from the existing datasets in our system to create a new dataset.
---

## 1. Adding Dataset
Make sure a dataset already exist and is showing in your dashboard. In below screenshot, a dataset already exist named **RAG data** and this is what we are going to be using for creating new dataset out of it.
![Step 1](./images/add-from-existing-dataset/step1.png)

## 2. Selecting Existing Dataset Option
Choose **Add from existing dataset** option to create dataset out of already existing dataset in dashboard.
![Step 2](./images/add-from-existing-dataset/step2.png)

## 3. Choose Dataset and Model Type
Assign **name** to this dataset and then **select the source dataset**. After that choose **model type**.
![Step 3](./images/add-from-existing-dataset/step3.png)

## 4. Dataset is Ready for Experimention
You can now see the copy of dataset on your dashboard. If not showing then refresh the page.
![Step 4](./images/add-from-existing-dataset/step4.png)
