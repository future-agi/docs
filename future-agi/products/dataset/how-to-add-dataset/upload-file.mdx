---
title: Uploading Files (JSON, CSV)
description: How to upload files in various formats.
---

## 1. Adding a Dataset
Select **Build** from the top left corner under the **Main** section. Then click on **Add Dataset**.
![Step 1](./images/upload-file/step1.png)

## 2. Selecting the Upload File Option
Choose the **Upload a file** option to upload files like CSV or JSON from your local computer.
![Step 2](./images/upload-file/step2.png)

## 3. Uploading a File
First, **name** your dataset and choose your **model type** from the drop-down menu, e.g., Generative LLM. Then **browse** for the file saved on your local computer that you want to upload. After selecting the file, click on **Done** and wait for it to upload.
![Step 3](./images/upload-file/step3.png)

## 4. Dataset is Ready for Experimentation
You can now see the uploaded dataset on your dashboard. If it is not showing, refresh the page.
![Step 4](./images/upload-file/step4.png)
