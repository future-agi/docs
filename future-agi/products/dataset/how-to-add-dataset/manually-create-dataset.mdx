This guide will walk you through the process of creating a new dataset manually in Future AGI.

## Steps to Create a Dataset

### 1. Access Dataset Creation
1. Navigate to the dashboard
2. Click the **Add Dataset** button in the top right corner
![Adding a new dataset](./images/manually-create-dataset/step1.png)

### 2. Select Creation Method
1. From the available options, choose **Manually Create Dataset**
2. This option allows you to create a new dataset from scratch
![Selecting manual creation method](./images/manually-create-dataset/step2.png)

### 3. Configure Dataset Settings
1. Enter a descriptive **name** for your dataset
2. Select the appropriate **model type** for your use case
3. Click **Create** to proceed
![Configuring dataset settings](./images/manually-create-dataset/step3.png)

### 4. Verify Dataset Creation
1. Once created, your new dataset will appear on the dashboard
2. You can now begin experimenting with your dataset
![Viewing the created dataset](./images/manually-create-dataset/step4.png)
