title: "Manually Create a Dataset"
---

The manual dataset creation feature allows users to define a dataset structure from scratch, adding rows and columns according to specific requirements using the enhanced Future AGI platform with improved type safety and structured data management.

---

### **Steps to Manually Create a Dataset**

### **1. Access the Dataset Creation Panel**

- Navigate to the **Dataset** section.
- Click the **"Add Dataset"** button to open the dataset creation options.
- Select **"Add Datasets Manually"** from the list.

### **2. Define the Dataset Structure**

A comprehensive setup interface appears with enhanced configuration options:

![manually_create_dataset](manually_creating.png)

#### **Basic Dataset Information**
- **Dataset Name**: Provide a descriptive name that clearly identifies the dataset purpose
- **Description**: Add detailed description for better dataset management
- **Model Type**: Select from available options:
  - **Generative LLM**: For text generation and language model tasks
  - **Classification**: For categorization and labeling tasks
  - **Regression**: For numerical prediction tasks
- **Tags**: Add relevant tags for easy discovery and organization

#### **Initial Structure Configuration**
- **Number of Rows**: Define the initial number of data entries to be created
- **Number of Columns**: Set up the data structure by specifying initial column count
- **Template Selection**: Choose from predefined templates:
  - **Conversational AI**: User query, context, response, rating
  - **Content Generation**: Prompt, generated content, quality score
  - **Classification**: Input text, category, confidence score
  - **Custom**: Define your own structure

### **3. Enhanced Column Management**

Once the dataset structure is defined, you can add and configure columns with advanced features:

#### **Access the Column Editor**
- Navigate to the **Data** tab of the newly created dataset
- Click the **"+ Add Column"** button in the table header
- The enhanced column editor opens with comprehensive configuration options

#### **Column Type Selection**

**Static Columns** (Manual Data Entry):
- **Text**: For textual data with configurable length limits
- **Integer**: For whole numbers with range validation
- **Float**: For decimal numbers with precision settings
- **Boolean**: For true/false values with custom labels
- **DateTime**: For timestamps with timezone support
- **JSON**: For structured data with schema validation
- **Array**: For list-type data with element type specification

**Dynamic Columns** (Automated Value Generation):
- **API Response**: Auto-populate from external API calls
- **Model Output**: Generated content from AI models
- **Calculated Fields**: Computed values based on other columns
- **System Metadata**: Timestamps, user IDs, session information

#### **Advanced Column Configuration**
- **Data Validation Rules**: Set constraints and validation criteria
- **Default Values**: Configure default values for new rows
- **Source Attribution**: Track data origin and lineage
- **Column Dependencies**: Define relationships between columns
- **Display Formatting**: Customize how data appears in the interface

### **4. SDK Integration for Manual Dataset Creation**

For programmatic dataset creation, use the enhanced SDK:

```python
from fi.datasets import Dataset
from fi.datasets.types import (
    DatasetConfig,
    Column,
    Row,
    Cell,
    DataTypeChoices,
    SourceChoices,
    ModelTypes
)

# Create dataset configuration
dataset_config = DatasetConfig(
    name="manual_customer_feedback",
    model_type=ModelTypes.GENERATIVE_LLM
)

# Initialize dataset
dataset = Dataset(dataset_config=dataset_config)
dataset = dataset.create()

# Define structured columns
columns = [
    Column(
        name="customer_query",
        data_type=DataTypeChoices.TEXT,
        source=SourceChoices.OTHERS
    ),
    Column(
        name="response_quality",
        data_type=DataTypeChoices.INTEGER,
        source=SourceChoices.OTHERS
    ),
    Column(
        name="response_helpful",
        data_type=DataTypeChoices.BOOLEAN,
        source=SourceChoices.OTHERS
    ),
    Column(
        name="created_at",
        data_type=DataTypeChoices.TEXT,  # Store as text for simplicity
        source=SourceChoices.OTHERS
    )
]

# Add columns to dataset
dataset = dataset.add_columns(columns=columns)

# Create sample rows
sample_rows = [
    Row(
        order=1,
        cells=[
            Cell(column_name="customer_query", value="How do I reset my password?"),
            Cell(column_name="response_quality", value=8),
            Cell(column_name="response_helpful", value=True),
            Cell(column_name="created_at", value="2024-01-15T10:30:00Z")
        ]
    ),
    Row(
        order=2,
        cells=[
            Cell(column_name="customer_query", value="What are your business hours?"),
            Cell(column_name="response_quality", value=9),
            Cell(column_name="response_helpful", value=True),
            Cell(column_name="created_at", value="2024-01-15T11:45:00Z")
        ]
    )
]

# Add rows to dataset
dataset = dataset.add_rows(rows=sample_rows)

print(f"Dataset created successfully with ID: {dataset.dataset_config.id}")
```

### **Best Practices for Manual Dataset Creation**

- **Plan Structure First**: Design your column schema before adding data
- **Use Descriptive Names**: Choose clear, meaningful column and dataset names
- **Implement Validation**: Set up data validation rules early in the process
- **Document Everything**: Add descriptions and metadata for future reference
- **Start Small**: Begin with a subset of data to validate your structure
- **Test Thoroughly**: Verify data quality and consistency before scaling
- **Version Control**: Maintain versions of your dataset as it evolves
