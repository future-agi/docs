title: "Identify Hallucination"
---


When AI models generate responses, sometimes the outputs stray from the original instructions or context, creating fabricated or irrelevant content. This phenomenon is known as **hallucination**.

Hallucinations can be frustrating, especially when the AI's reliability matters. Whether you're building a chatbot, summarising documents, or generating answers, identifying hallucinations is key to improving accuracy and trust in your AI system.

Hallucinations happen when the output:

- Doesn't follow the user's instructions.
- Introduces information that isn't part of the given context.
- Strays into unrelated topics or makes unsupported claims.

That is why detecting hallucination matters because it ensures accuracy by reducing errors in AI-generated content, which is critical for applications like customer service, research, or education and it can help builds trust with its consistent and grounded responses, thus making users confident in the AI system's capabilities.

Following are the evals to identify hallucinations in AI-generated text content:

- [Prompt/Instruction Adherence](#1-promptinstruction-adherence)
- [Context Adherence](#2-context-adherence)
- [Factual Accuracy](#3-factual-accuracy)

## 2. Context Adherence

Evaluates how well responses stay within the provided context by measuring if the output contains any information not present in the given context.

[Click here to read the eval definition of Context Adherence](/future-agi/products/evaluation/eval-definition/context-adherence)

### a. Using Interface


**Inputs Required**:

- **Output**: The response generated by the AI system.
- **Context**: The background information or source material the response should be based on.

### b. Using SDK

```python
#context adherence
from fi.testcase import TestCase
from fi.evals import ContextAdherence

adherence_scores = []
adherence_reasons = []
for _, row in dataset.iterrows():
    test_case = TestCase(
        output=row["question"],
        context=row["context"]
    )
    adherence_template = ContextAdherence(config={
        "required_keys": ["output", "context"],
        "output": "adherence_score",
    })

    response = evaluator.evaluate(eval_templates=[adherence_template], inputs=[test_case])

    adherence_result = response.eval_results[0].metrics[0].value
    reason = response.eval_results[0].reason

    adherence_scores.append(adherence_result)
    adherence_reasons.append(reason)

dataset["context_adherence"] = adherence_scores
dataset["adherence_reason"] = adherence_reasons

```
