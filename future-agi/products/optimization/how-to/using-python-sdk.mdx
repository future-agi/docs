## **1. Installation and Setup**

Before proceeding with optimization, ensure that the Future AGI SDK is installed and properly configured with your API credentials.

**Installation**

```bash
pip install futureagi
```

**Set up API credentials**

Set your API credentials as environment variables:

```bash
export FI_API_KEY="your_api_key"
export FI_SECRET_KEY="your_secret_key"
```

Or set them programmatically:

```python
import os
os.environ["FI_API_KEY"] = "your_api_key"
os.environ["FI_SECRET_KEY"] = "your_secret_key"
```

## **2. Creating a Dataset**

Optimization requires a structured dataset that serves as input for generating and refining AI responses. If you don't have a dataset yet, follow these steps to create one.

### **Initialize the Dataset**

The `Dataset` class manages dataset creation and operations. First, define the dataset properties:

```python
from fi.datasets import Dataset
from fi.datasets.types import (
    DatasetConfig, Column, Row, Cell, 
    DataTypeChoices, SourceChoices, ModelTypes
)

# Define dataset configuration
dataset_config = DatasetConfig(
    name="optimization_dataset",
    model_type=ModelTypes.GENERATIVE_LLM
)

# Initialize dataset instance
dataset = Dataset(dataset_config=dataset_config)
```

### **Creating an Empty Dataset**

Create an empty dataset and add structured data:

```python
# Create the dataset
dataset = dataset.create()
print(f"Dataset created: {dataset.dataset_config.name}")

# Define columns
columns = [
    Column(
        name="value_proposition",
        data_type=DataTypeChoices.TEXT,
        source=SourceChoices.OTHERS
    ),
    Column(
        name="context",
        data_type=DataTypeChoices.TEXT,
        source=SourceChoices.OTHERS
    )
]

# Add columns to dataset
dataset = dataset.add_columns(columns=columns)

# Add sample data
rows = [
    Row(
        order=1,
        cells=[
            Cell(column_name="value_proposition", value="Our AI platform helps businesses automate customer support"),
            Cell(column_name="context", value="Technology startup focused on customer service automation")
        ]
    ),
    Row(
        order=2,
        cells=[
            Cell(column_name="value_proposition", value="Sustainable energy solutions for modern homes"),
            Cell(column_name="context", value="Green energy company specializing in residential solar")
        ]
    )
]

dataset = dataset.add_rows(rows=rows)
print("Dataset populated with sample data")
```

### **Creating a Dataset from a File**

If you have a dataset in a **CSV, JSON, or Excel file**, you can create it directly:

```python
from fi.datasets.types import HuggingfaceDatasetConfig

# Create from local file
dataset = Dataset.create_dataset(
    dataset_config=dataset_config,
    source="data.csv"  # Path to your local file
)

# Or create from Hugging Face
hf_config = HuggingfaceDatasetConfig(
    name="squad",
    subset="v2.0",
    split="train",
    num_rows=50
)

dataset = Dataset.create_dataset(
    dataset_config=dataset_config,
    source=hf_config
)
```

## **3. Running a Prompt on the Dataset**

Before optimizing a prompt, you need to define a **baseline prompt**. This serves as the **starting point for evaluation**.

```python
# Add a run prompt column
messages = [
    {
        "role": "system",
        "content": "You are a marketing expert. Create compelling summaries."
    },
    {
        "role": "user",
        "content": "Create a short summary for this value proposition: {{value_proposition}}"
    }
]

dataset = dataset.add_run_prompt(
    name="summary_prompt",
    model="gpt-4o-mini",
    messages=messages,
    temperature=0.7,
    max_tokens=150
)

print("Baseline prompt added and executed")
```

- `add_run_prompt()` **adds a new prompt column to the dataset**.
- `messages` define **how the AI is instructed**, with `{{value_proposition}}` acting as a **placeholder** for dataset values.
- The prompt will be executed on all rows in the dataset, generating responses.

---

## **4. Evaluating AI Responses**

Once the prompt has been run, we need to measure **how effective the responses are** using evaluation metrics.

```python
# Add evaluation to assess response quality
dataset = dataset.add_evaluation(
    name="response_quality_check",
    eval_template="LengthGreaterThan",
    required_keys_to_column_names={
        "text": "summary_prompt"  # Column to evaluate
    },
    config={"min_length": 20},  # Minimum length requirement
    run=True  # Execute evaluation immediately
)

print("Evaluation added and executed")

# Wait for evaluation to process
import time
time.sleep(5)

# Check evaluation results
try:
    eval_stats = dataset.get_eval_stats()
    print(f"Evaluation completed. Found {len(eval_stats)} metrics.")
except Exception as e:
    print(f"Could not retrieve evaluation stats: {e}")
```

- `add_evaluation()` attaches an evaluation metric to analyze responses.
- `"LengthGreaterThan"` is a built-in evaluation template that checks text length.
- `required_keys_to_column_names` maps evaluation inputs to dataset columns.
- `run=True` executes the evaluation immediately.

### **Available Evaluation Templates**

Common evaluation templates include:
- `"LengthGreaterThan"` - Checks if text meets minimum length
- `"Contains"` - Checks if text contains specific substrings
- `"Tone"` - Analyzes text tone and sentiment
- `"Relevance"` - Measures relevance to input context

## **5. Running Optimization**

Optimization improves AI-generated responses by adjusting prompt structure based on evaluation feedback. This process systematically **iterates through prompt variations to find the most effective version**.

```python
# Setup optimization with enhanced configuration
try:
    dataset = dataset.add_optimization(
        optimization_name="summary_prompt_optimization",
        prompt_column_name="summary_prompt",  # Column to optimize
        optimize_type="PROMPT_TEMPLATE",
        model_config={
            "model_name": "gpt-4o-mini",
            "temperature": 0.3,
            "max_tokens": 200,
            "frequency_penalty": 0.0,
            "presence_penalty": 0.0,
            "top_p": 1.0
        }
    )
    print("Optimization setup completed successfully!")
    
except Exception as e:
    print(f"Optimization setup failed: {e}")
    print("Ensure evaluations are properly configured and processed.")
```

- `add_optimization()` starts an optimization process.
- `"summary_prompt_optimization"` assigns a name to the optimization.
- `"summary_prompt"` specifies which prompt column is being optimized.
- `model_config` provides detailed model parameters for optimization.

### **Optimization Parameters**

Key parameters for optimization:
- `optimization_name`: Unique identifier for the optimization
- `prompt_column_name`: The prompt column to optimize
- `optimize_type`: Type of optimization ("PROMPT_TEMPLATE", "MODEL_PARAMETERS", "HYBRID")
- `model_config`: Model configuration including temperature, max_tokens, etc.

## **7. Complete Optimization Workflow Example**

Here's a complete example that demonstrates the entire optimization workflow:

```python
import os
import time
from fi.datasets import Dataset
from fi.datasets.types import (
    DatasetConfig, Column, Row, Cell, 
    DataTypeChoices, SourceChoices, ModelTypes
)

# Set up environment
os.environ["FI_API_KEY"] = "your_api_key"
os.environ["FI_SECRET_KEY"] = "your_secret_key"

def run_optimization_workflow():
    try:
        # 1. Create dataset
        config = DatasetConfig(
            name="marketing_optimization_example",
            model_type=ModelTypes.GENERATIVE_LLM
        )
        
        dataset = Dataset(dataset_config=config)
        dataset = dataset.create()
        print(f"✓ Dataset created: {dataset.dataset_config.name}")
        
        # 2. Add data structure
        columns = [
            Column(name="product_description", data_type=DataTypeChoices.TEXT, source=SourceChoices.OTHERS),
            Column(name="target_audience", data_type=DataTypeChoices.TEXT, source=SourceChoices.OTHERS)
        ]
        dataset = dataset.add_columns(columns=columns)
        
        # 3. Add sample data
        rows = [
            Row(order=1, cells=[
                Cell(column_name="product_description", value="AI-powered project management tool"),
                Cell(column_name="target_audience", value="Software development teams")
            ]),
            Row(order=2, cells=[
                Cell(column_name="product_description", value="Eco-friendly cleaning products"),
                Cell(column_name="target_audience", value="Environmentally conscious families")
            ])
        ]
        dataset = dataset.add_rows(rows=rows)
        
        # 4. Add baseline prompt
        messages = [
            {
                "role": "system",
                "content": "You are a marketing copywriter. Create compelling product summaries."
            },
            {
                "role": "user",
                "content": "Product: {{product_description}}\nTarget: {{target_audience}}\n\nCreate a compelling 2-sentence marketing summary:"
            }
        ]
        
        dataset = dataset.add_run_prompt(
            name="marketing_summary",
            model="gpt-4o-mini",
            messages=messages,
            temperature=0.7,
            max_tokens=100
        )
        
        # 5. Add evaluation
        dataset = dataset.add_evaluation(
            name="summary_quality",
            eval_template="LengthGreaterThan",
            required_keys_to_column_names={"text": "marketing_summary"},
            config={"min_length": 30},
            run=True
        )
        
        # 6. Wait for processing
        print("Waiting for evaluation to process...")
        time.sleep(10)
        
        # 7. Run optimization
        dataset = dataset.add_optimization(
            optimization_name="marketing_prompt_optimization",
            prompt_column_name="marketing_summary",
            optimize_type="PROMPT_TEMPLATE",
            model_config={
                "model_name": "gpt-4o-mini",
                "temperature": 0.5,
                "max_tokens": 150,
                "frequency_penalty": 0.1,
                "presence_penalty": 0.1,
                "top_p": 0.9
            }
        )
        print("✓ Optimization completed successfully!")
        
        # 8. Download results
        df = dataset.download(load_to_pandas=True)
        print(f"Results shape: {df.shape}")
        
    except Exception as e:
        print(f"Optimization failed: {e}")
    
    finally:
        # Cleanup
        if 'dataset' in locals() and dataset.dataset_config:
            dataset.delete()
            print("Dataset cleaned up")

# Run the workflow
if __name__ == "__main__":
    run_optimization_workflow()
```

## **8. Best Practices for Optimization**

### **Dataset Preparation**
- Ensure your dataset has sufficient variety in input data
- Include representative examples of your use case
- Use clear, descriptive column names

### **Prompt Design**
- Start with a clear, well-structured baseline prompt
- Use appropriate placeholders for dynamic content
- Consider the target audience and desired output format

### **Evaluation Strategy**
- Choose evaluation metrics that align with your goals
- Allow sufficient time for evaluation processing
- Monitor evaluation results before running optimization

### **Optimization Configuration**
- Start with conservative model parameters
- Gradually adjust temperature and other settings
- Use appropriate optimization types for your use case

### **Error Handling**
- Always include proper error handling in your code
- Check evaluation status before running optimization
- Implement retry logic for timing-sensitive operations

## **9. Troubleshooting Common Issues**

### **"No evaluation templates found" Error**
This error occurs when optimization cannot find evaluations for the specified column:

```python
# Solution: Ensure evaluations are properly configured and processed
try:
    # Check evaluation status
    eval_stats = dataset.get_eval_stats()
    print(f"Available evaluations: {len(eval_stats)}")
    
    # Wait longer for processing
    time.sleep(10)
    
    # Retry optimization
    dataset.add_optimization(
        optimization_name="retry_optimization",
        prompt_column_name="summary_prompt",
        optimize_type="PROMPT_TEMPLATE",
        model_config={
            "model_name": "gpt-4o-mini",
            "temperature": 0.3,
            "max_tokens": 200
        }
    )
    
except Exception as e:
    print(f"Error: {e}")
    print("Ensure evaluations reference the correct column names")
```

### **Authentication Issues**
```python
# Verify credentials are set
import os
print("API Key set:", bool(os.getenv("FI_API_KEY")))

```

### **Timing Issues**
```python

import time

```

### **Column Reference Issues**
```python
# Verify column exists before optimization
column_id = dataset.get_column_id("summary_prompt")

    print(f"Column found with ID: {column_id}")

```
