---
title: Prompt Evaluation
description: 'Evaluate your prompts on their ability to generate the desired output.'
---

Prompt evaluation assesses how effectively a prompt generates the desired output based on specified evaluation criteria. This evaluation type is particularly useful when you need to validate whether your prompts are well-designed and produce appropriate responses.

### Required Parameters

| Parameter | Description | Required |
|-----------|-------------|-----------|
| `input` | The input text to be evaluated | Yes |
| `output` | The output generated by the model | Yes |
| `prompt` | The prompt used to generate the output | Yes |

### Configuration

The evaluation accepts the following configuration:

| Parameter | Description | Required |
|-----------|-------------|-----------|
| `criteria` | The evaluation criteria | Yes |

### Example

```python
from fi.evals import PromptAdherence, EvalClient
from fi.testcases import LLMTestCase

# Initialize the evaluation client
evaluator = EvalClient(
    fi_api_key="your_api_key",
    fi_secret_key="your_secret_key"
)

# Create a test case
test_case = LLMTestCase(
    input="What is the capital of France?",
    output="Paris is the capital city of France.",
    prompt="Answer the following question accurately and concisely:"
)

# Initialize the prompt evaluator
prompt_eval = PromptAdherence(
    config = {
        "criteria": """
        Evaluate the prompt based on:
        - Clarity of instructions
        - Specificity of requirements
        - Appropriate constraints
        - Output format guidance
        - Context relevance
        """
    }
)

# Run the evaluation
result = evaluator.evaluate(prompt_eval, test_case)
print(result) # Will return a score between 0 and 1
```
