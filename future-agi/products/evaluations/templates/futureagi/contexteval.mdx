title: Context Evaluation

Context evaluation assesses how effectively a given context generates the desired output based on specified evaluation criteria. This evaluation type is particularly useful when you need to validate whether the provided context contains sufficient and relevant information to produce accurate and appropriate responses.

| Parameter | Description | Required |
|-----------|-------------|-----------|
| `input` | The input context to be evaluated | Yes |
| `output` | The output generated by the model | Yes |
| `context` | The context used to generate the output | Yes |



### Configuration
The evaluation requires no configuration:

| Parameter | Description | Required |
|-----------|-------------|-----------|
| `criteria` | The evaluation criteria | Yes |

```python
from fi.evals import ContextEvaluator

# Initialize the context evaluator
context_eval = ContextEvaluator(config = {
    "criteria": "Evaluate based on: Context relevance to input Output accuracy based on context Completeness of information Proper use of provided context"
})
```


### Test Case Setup
The evaluation requires the input, output, context and evaluation criteria:


```python
from fi.testcases import LLMTestCase

test_case = LLMTestCase(
    input="What is the capital of France?",
    output="Paris is the capital city of France.",
    context="Paris is the capital and largest city of France, situated in the north-central part of the country.",
)
```
### Client Setup
Initialize the evaluation client with your API credentials:

```python
# Initialize the evaluation client
evaluator = EvalClient(
    fi_api_key="your_api_key",
    fi_secret_key="your_secret_key"
)
```

### Complete Example

```python
from fi.evals import ContextEvaluator, EvalClient
from fi.testcases import LLMTestCase

# Initialize the context evaluator
context_eval = ContextEvaluator(
    config = {
        "criteria": "Evaluate based on: Context relevance to input Output accuracy based on context Completeness of information Proper use of provided context"
    }
)

# Create a test case
test_case = LLMTestCase(
    input="What is the capital of France?",
    output="Paris is the capital city of France.",
    context="Paris is the capital and largest city of France, situated in the north-central part of the country.",
)

# Initialize the evaluation client
evaluator = EvalClient(
    fi_api_key="your_api_key",
    fi_secret_key="your_secret_key"
)

# Run the evaluation
result = evaluator.evaluate(context_eval, test_case)
print(result)
```
