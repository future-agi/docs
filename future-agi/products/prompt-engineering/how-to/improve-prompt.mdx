---
title: "Improve an Existing Prompt"
---

Improving an existing prompt enables users to refine prompts for clarity, precision, and effectiveness. This feature allows for structured modifications to enhance the prompt’s readability, tone, and logical structure while maintaining its intended functionality.

---

### 1. Accessing the Prompt Improvement Interface

1. Navigate to **Prompt Management** from the left-hand menu.
2. Click **Improve an Existing Prompt** to enter the prompt enhancement interface.
3. This section allows users to refine prompts by applying AI-driven improvements.

### 2. Providing an Existing Prompt

1. In the **Enter User Prompt Here** field, input the prompt that needs refinement.
   - Example: *"Write an email."*
2. This prompt serves as the base that the AI will analyze and improve.

### 3. Specifying Improvements

1. In the **What would you like to Improve?** field, describe the specific enhancement required.
   - Example: *"Make the tone professional."*
2. This step ensures targeted refinements based on user requirements.
3. Click **Submit** to allow the AI to process and refine the prompt.

### 4. Reviewing the Improved Prompt

1. The AI-generated refinement appears in the **Results** section.
2. The refinement process typically includes:
   - Enhanced clarity and logical flow.
   - Improved grammatical consistency.
   - Optimization for specificity and effectiveness.
3. Users can assess the refinements and make additional modifications if necessary.
4. Click **Apply** to finalize the improved prompt.

---

Future AGI provides a structured process for refining existing prompts, ensuring clarity, effectiveness, and coherence. By leveraging AI-driven improvements, users can create high-quality prompts tailored to their specific needs.
