---
title: Generate a Prompt
---

This feature simplifies prompt design by analyzing user instructions and dynamically generating a well-structured prompt.

## 1. Accessing the Prompt Generation Interface
- Navigate to **Prompt Management** from the left-hand menu.
- Click **Generate a Prompt** to enter the AI-assisted prompt creation section.
- The interface provides a structured workflow where users can describe their intended prompt.

## 2. Providing a Prompt Input
- In the **Describe your prompt** field, enter a general task description.
  - Example: *"Write me an email."*
- Select from predefined categories if applicable:
  - Options include **Summarize a document, Translate code, Content moderation, Recommend a product, etc.**
- Click **Generate** to allow the AI system to create a structured prompt based on your input.

## 3. Reviewing the Generated Prompt
- The AI generates a structured response in the **Results** section.
- The structured prompt typically includes:
  - A clear task definition.
  - Placeholder variables (e.g., `{{SUBJECT}}`, `{{INTRODUCTION}}`).
  - Logical flow and chain-of-thought instructions for AI understanding.
- Review the AI-generated prompt to ensure clarity and accuracy.
- Click **Apply** to finalize the prompt for use.

## 4. Refining and Customizing the Prompt
- Use the **Follow-up Prompt** field to modify the generated prompt.
- Suggested improvements may include:
  - Enhancing clarity.
  - Adjusting grammar and structure.
  - Fine-tuning the output format.
- Click **Apply** to confirm and integrate the modifications.

---
Future AGI’s automated prompt generation feature simplifies the process of creating effective AI prompts. By leveraging structured workflows, predefined templates, and evaluation tools, users can efficiently generate, refine, and manage high-quality prompts for various applications.
