1. Installation
Install the FutureAGI package to access the observability framework.

```bash
pip install futureagi
```

---

## 2. Environment Configuration
Set up your environment variables to authenticate with FutureAGI services. These credentials enable:
- Authentication with FutureAGI's observability platform
- Encrypted telemetry data transmission

```python
import os
os.environ["FI_API_KEY"] = "your-futureagi-api-key"
os.environ["FI_SECRET_KEY"] = "your-futureagi-secret-key"
```


---
## 3. Configure Evaluation Tags
Define evaluation criteria for monitoring LLM responses. Evaluation tags allow you to:
- Define custom evaluation criteria
- Set up automated response quality checks
- Track model performance metrics

> Click here [here](/future-agi/products/observe/how-to/evals) to learn how to configure eval tags for observability.

```python
from fi_instrumentation.types import EvalName, EvalSpanKind, EvalTag, EvalTagType

eval_tags = [
    EvalTag(
        eval_name=EvalName.DETERMINISTIC_EVALS,
        value=EvalSpanKind.TOOL,
        type=EvalTagType.OBSERVATION_SPAN,
        config={
            "multi_choice": False,
            "choices": ["Yes", "No"],
            "rule_prompt": "Evaluate if the response is correct",
        },
        custom_eval_name="det_eval_crewai_1"
    )
]
```

---
## 4. Initialize Trace Provider
Set up the trace provider to establish the observability pipeline. The trace provider:
- Creates a new project in FutureAGI
- Establishes telemetry data pipelines
- Configures version tracking
- Sets up evaluation frameworks

```python
from fi_instrumentation import register
from fi_instrumentation.types import ProjectType

trace_provider = register(
    project_type=ProjectType.EXPERIMENT,
    project_name="crew_ai_app",
    project_version_name="v1",
    eval_tags=eval_tags
)
```

---
## 5. Configure Crew AI Instrumentation
Initialize the Crew AI instrumentor to enable automatic tracing.

```python   
from traceai_crewai import CrewAIInstrumentor

CrewAIInstrumentor().instrument(tracer_provider=trace_provider)
```

## 7. Create Crew AI Components
Set up your Crew AI components with built-in observability.

```python
from crewai import LLM, Agent, Crew, Process, Task
from crewai_tools import SerperDevTool

def story_example():
    llm = LLM(
        model="gpt-4",
        temperature=0.8,
        max_tokens=150,
        top_p=0.9,
        frequency_penalty=0.1,
        presence_penalty=0.1,
        stop=["END"],
        seed=42,
    )

    writer = Agent(
        role="Writer",
        goal="Write creative stories",
        backstory="You are a creative writer with a passion for storytelling",
        allow_delegation=False,
        llm=llm,
    )

    writing_task = Task(
        description="Write a short story about a magical forest",
        agent=writer,
        expected_output="A short story about a magical forest",
    )

    crew = Crew(agents=[writer], tasks=[writing_task])

    # Execute the crew
    result = crew.kickoff()
    print(result)
```

---
## 8. Execute
Run your Crew AI application.

```python
if __name__ == "__main__":
    story_example()
```
