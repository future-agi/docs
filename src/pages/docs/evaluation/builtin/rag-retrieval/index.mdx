## RAG & retrieval

Evals in this category support **RAG and retrieval**: whether the model uses retrieved chunks, how well it leverages them, recall of reference information, and quality of retrieved context.

| Eval | Description | Inputs | Output | Use case |
| --- | --- | --- | --- | --- |
| Chunk Attribution | Evaluates whether the model references the provided context chunks when generating a response. | `context`, `output` | Pass/Fail | Checking if RAG context is used. |
| Chunk Utilization | Measures how effectively the model uses information from context to produce coherent output. | `context`, `output` | Score | Quality of context use in RAG. |
| Recall Score | Measures how much of the reference information is captured in the hypothesis. | `hypothesis`, `reference` | Score | Retrieval or generation recall. |
| Eval Context Retrieval | Evaluates the quality of context retrieved for generating a response. | `input`, `context` (or similar) | Score | RAG retrieval pipeline quality. |
