---
title: "Is Sql: Built-in Evaluation"
description: "Validates if text appears to be syntactically valid SQL. Checks for recognized SQL keywords, balanced parentheses, and balanced quotes"
---

<CodeGroup>

```python Python
result = evaluator.evaluate(
    eval_templates="is_sql",
    inputs={
        "text": "Hello, this is a sample text."
    },
)

print(result.eval_results[0].output)
print(result.eval_results[0].reason)
```

```typescript JS/TS
import { Evaluator } from "@future-agi/ai-evaluation";

const evaluator = new Evaluator();

const result = await evaluator.evaluate(
  "is_sql",
  {
    text: "Hello, this is a sample text."
  }
);

console.log(result);
```

| **Input** |  |  |  |

| | **Required Input** | **Type** | **Description** |

| **Output** |  |  |
