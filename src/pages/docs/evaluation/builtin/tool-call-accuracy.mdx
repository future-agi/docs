Evaluates accuracy of agent tool/function calls by comparing actual vs expected calls. Checks function names and arguments, supporting both OpenAI and generic tool call formats. Scores full matches (name+args) at 1.0 and name-only matches at 0.5.

<CodeGroup>

```python Python
result = evaluator.evaluate(
    eval_templates="tool_call_accuracy",
    inputs={
        "output": "The capital of France is Paris.",
        "expected": "Paris"
    },
)

print(result.eval_results[0].output)
print(result.eval_results[0].reason)
```

```typescript JS/TS
import { Evaluator } from "@future-agi/ai-evaluation";

const evaluator = new Evaluator();

const result = await evaluator.evaluate(
  "tool_call_accuracy",
  {
    output: "The capital of France is Paris.",
    expected: "Paris"
  }
);

console.log(result);
```

</CodeGroup>

| **Input** |  |  |  |
| ------ | --------- | ---- | ----------- |
| | **Required Input** | **Type** | **Description** |
| | `output` | `string` | The output. |
| | `expected` | `string` | The expected. |

| **Output** |  |  |
| ------ | ----- | ----------- |
| | **Field** | **Description** |
| | **Result** | Returns a numeric score between 0 and 1, plus a reason explaining the verdict. |
| | **Reason** | A plain-language explanation of the verdict. |
