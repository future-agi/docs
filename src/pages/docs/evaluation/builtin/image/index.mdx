## Image

Evals in this category work on **image** (or image + text) inputs. Use them to detect caption hallucination, judge synthetic vs real images, or score how well an image matches instructions.

| Eval | Description | Inputs | Output | Use case |
| --- | --- | --- | --- | --- |
| Caption Hallucination | Evaluates whether an image caption contains fabricated information not visible in the image. | `image`, `caption` | Pass/Fail | Detecting hallucinated details in image captions. |
| Synthetic Image Evaluator | Evaluates whether an image was AI-generated or camera/human-created. | `image` | Score | Detecting synthetic vs real images. |
| Eval Image Instruction | Scores how well the image reflects given textual instructions. | `input`, `image_url` (or similar) | Score | Checking instruction–image alignment. |
