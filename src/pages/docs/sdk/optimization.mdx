<TLDR>
- `pip install agent-opt` — separate package, depends on `ai-evaluation`
- 6 algorithms: Random Search, Bayesian, ProTeGi, Meta-Prompt, PromptWizard, GEPA
- Uses eval metrics as the scoring function to find the best prompt
</TLDR>

agent-opt finds the best prompt for your task automatically. You give it a starting prompt, a dataset, and a scoring metric. It generates variations, scores them, and returns the highest-performing prompt.

<Note>
  Requires `pip install agent-opt`. This also installs `ai-evaluation` and `futureagi` as dependencies.
</Note>

## Quick Example

```python
from fi.opt.optimizers import RandomSearchOptimizer
from fi.opt.generators import LiteLLMGenerator
from fi.opt.base.evaluator import Evaluator
from fi.opt.datamappers import BasicDataMapper
from fi.evals.metrics.heuristics.string import ContainsMetric

# 1. Your dataset
dataset = [
    {"article": "Python 3.12 adds better error messages...", "expected": "Python 3.12 improves errors"},
    {"article": "React Server Components reduce bundle size...", "expected": "RSC cuts bundle size"},
]

# 2. Generator — the LLM + prompt template to optimize
generator = LiteLLMGenerator(
    model="gpt-4o-mini",
    prompt_template="Summarize in one sentence: {article}",
)

# 3. Evaluator — how to score each output
evaluator = Evaluator(metric=ContainsMetric())

# 4. Data mapper — connects generator output to evaluator inputs
data_mapper = BasicDataMapper(key_map={"output": "generated_output", "expected_output": "expected"})

# 5. Optimize
optimizer = RandomSearchOptimizer(generator=generator, teacher_model="gpt-4o", num_variations=5)
result = optimizer.optimize(evaluator=evaluator, data_mapper=data_mapper, dataset=dataset)

print(f"Best prompt: {result.best_generator.prompt_template}")
print(f"Score: {result.final_score}")
```

## Algorithms

| Algorithm | Best for | How it works |
|-----------|----------|-------------|
| `RandomSearchOptimizer` | Quick baselines | Generates random prompt variations |
| `BayesianSearchOptimizer` | Few-shot tuning | Optuna-powered parameter search |
| `ProTeGi` | Iterative refinement | Textual gradients — analyzes failures and rewrites |
| `MetaPromptOptimizer` | Teacher-driven | A stronger model analyzes and rewrites the prompt |
| `PromptWizardOptimizer` | Multi-stage refinement | Mutation → critique → refine pipeline |
| `GEPAOptimizer` | Complex search spaces | Genetic Pareto evolutionary optimization |

```python
from fi.opt.optimizers import (
    RandomSearchOptimizer,
    BayesianSearchOptimizer,
    ProTeGi,
    MetaPromptOptimizer,
    PromptWizardOptimizer,
    GEPAOptimizer,
)
```

All optimizers share the same interface:

```python
optimizer = SomeOptimizer(generator=generator, teacher_model="gpt-4o", num_variations=5)
result = optimizer.optimize(evaluator=evaluator, data_mapper=data_mapper, dataset=dataset)
```

## Core Components

### Generator

Wraps an LLM and a prompt template. The optimizer modifies the prompt template to improve scores.

```python
from fi.opt.generators import LiteLLMGenerator

generator = LiteLLMGenerator(
    model="gpt-4o-mini",
    prompt_template="Summarize this article: {article}",
)
```

Use `{field_name}` placeholders to reference dataset fields.

### Evaluator

Scores each generated output. Supports heuristic metrics, LLM-as-Judge, or Turing cloud evals.

```python
from fi.opt.base.evaluator import Evaluator
from fi.evals.metrics.heuristics.string import ContainsMetric

# Heuristic metric
evaluator = Evaluator(metric=ContainsMetric())

# Turing cloud eval
evaluator = Evaluator(eval_template="is_helpful", eval_model_name="turing_flash")

# LLM-as-Judge (via custom metric)
from fi.evals.metrics.llm_as_judges.custom_judge.metric import CustomLLMJudge
judge = CustomLLMJudge(provider=LiteLLMProvider(), config={"grading_criteria": "...", "model": "gpt-4o"})
evaluator = Evaluator(metric=judge)
```

### Data Mapper

Maps generator output + dataset fields to evaluator input format.

```python
from fi.opt.datamappers import BasicDataMapper

mapper = BasicDataMapper(key_map={
    "output": "generated_output",      # generator output → "output" for evaluator
    "expected_output": "expected",      # dataset field "expected" → "expected_output"
})
```

### Result

The `optimize()` call returns a result with the best prompt and score history.

```python
result = optimizer.optimize(...)

print(result.best_generator.prompt_template)  # the winning prompt
print(result.final_score)                      # best score achieved
print(result.history)                          # score history across iterations
```
