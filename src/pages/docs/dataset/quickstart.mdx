## What you'll do

By the end of this guide you will have:

1. Installed the FutureAGI SDK
2. Created a dataset
3. Added columns and rows
4. Verified your data in the dashboard

---

## How to

<Steps>
  <Step title="Install the SDK">
    <CodeGroup>
    ```bash Python
    pip install futureagi
    ```
    ```bash JS/TS
    npm install @futureagi/sdk
    ```
    </CodeGroup>
  </Step>

  <Step title="Set your API keys">
    Get your keys from the [dashboard](https://app.futureagi.com/dashboard/keys).

    <CodeGroup>
    ```python Python
    import os
    os.environ["FI_API_KEY"] = "YOUR_API_KEY"
    os.environ["FI_SECRET_KEY"] = "YOUR_SECRET_KEY"
    ```
    ```typescript JS/TS
    process.env.FI_API_KEY = "YOUR_API_KEY";
    process.env.FI_SECRET_KEY = "YOUR_SECRET_KEY";
    ```
    </CodeGroup>
  </Step>

  <Step title="Create a dataset">
    <CodeGroup>
    ```python Python
    from fi.datasets import Dataset
    from fi.datasets.types import DatasetConfig, ModelTypes

    config = DatasetConfig(name="my-first-dataset", model_type=ModelTypes.GENERATIVE_LLM)
    dataset = Dataset(dataset_config=config)
    dataset = dataset.create()
    print("Dataset created:", dataset.name)
    ```
    ```typescript JS/TS
    import { Dataset } from "@futureagi/sdk";

    const dataset = await Dataset.open("my-first-dataset");
    console.log("Dataset created:", dataset.name);
    ```
    </CodeGroup>
  </Step>

  <Step title="Add columns and rows">
    <CodeGroup>
    ```python Python
    from fi.datasets.types import Cell, Column, Row, DataTypeChoices, SourceChoices

    columns = [
        Column(name="user_query",  data_type=DataTypeChoices.TEXT,    source=SourceChoices.OTHERS),
        Column(name="llm_response", data_type=DataTypeChoices.TEXT,   source=SourceChoices.OTHERS),
        Column(name="score",        data_type=DataTypeChoices.INTEGER, source=SourceChoices.OTHERS),
    ]

    rows = [
        Row(order=1, cells=[
            Cell(column_name="user_query",   value="What is machine learning?"),
            Cell(column_name="llm_response", value="Machine learning is a subset of AI..."),
            Cell(column_name="score",        value=9),
        ]),
        Row(order=2, cells=[
            Cell(column_name="user_query",   value="Explain transformers"),
            Cell(column_name="llm_response", value="Transformers are a neural network architecture..."),
            Cell(column_name="score",        value=8),
        ]),
    ]

    dataset = dataset.add_columns(columns=columns)
    dataset = dataset.add_rows(rows=rows)
    print("✓ Data added successfully")
    ```
    ```typescript JS/TS
    import { DataTypeChoices, createRow, createCell } from "@futureagi/sdk";

    await dataset.addColumns([
        { name: "user_query",   dataType: DataTypeChoices.TEXT },
        { name: "llm_response", dataType: DataTypeChoices.TEXT },
        { name: "score",        dataType: DataTypeChoices.INTEGER },
    ]);

    await dataset.addRows([
        createRow({ cells: [
            createCell({ columnName: "user_query",   value: "What is machine learning?" }),
            createCell({ columnName: "llm_response", value: "Machine learning is a subset of AI..." }),
            createCell({ columnName: "score",        value: 9 }),
        ]}),
        createRow({ cells: [
            createCell({ columnName: "user_query",   value: "Explain transformers" }),
            createCell({ columnName: "llm_response", value: "Transformers are a neural network architecture..." }),
            createCell({ columnName: "score",        value: 8 }),
        ]}),
    ]);

    console.log("✓ Data added successfully");
    ```
    </CodeGroup>
  </Step>

  <Step title="View your dataset">
    Open the [Datasets dashboard](https://app.futureagi.com/dashboard/develop) to see your new dataset with all its rows and columns.
  </Step>
</Steps>
