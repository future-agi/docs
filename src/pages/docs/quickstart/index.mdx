<TLDR>
Install the SDK, set your API key, and evaluate your first LLM output in 3 steps:
- `pip install futureagi`
- Set `FUTUREAGI_API_KEY` environment variable
- Call `client.evaluate()` with your input/output
</TLDR>

<Prerequisites>
- Python 3.8+
- A Future AGI account ([sign up free](https://app.futureagi.com/signup))
- Your API key from the [dashboard](https://app.futureagi.com/settings/api-keys)
</Prerequisites>

## Install the SDK

```bash
pip install futureagi
```

For optional framework integrations:

```bash
pip install futureagi[langchain]  # LangChain support
pip install futureagi[llamaindex]  # LlamaIndex support
pip install futureagi[all]         # All integrations
```

## Set your API key

<CodeGroup titles={["Environment Variable", "Direct"]}>
  <CodePanel index={0}>
    ```bash
    export FUTUREAGI_API_KEY="fagi_xxxxxxxxxxxx"
    ```
  </CodePanel>
  <CodePanel index={1}>
    ```python
    from futureagi import FutureAGI

    client = FutureAGI(api_key="fagi_xxxxxxxxxxxx")
    ```
  </CodePanel>
</CodeGroup>

<Callout type="tip">
Store your API key in a `.env` file and use `python-dotenv` to load it.
</Callout>

## Run your first evaluation

Evaluate any LLM output with built-in metrics:

```python
from futureagi import FutureAGI

client = FutureAGI()

result = client.evaluate(
    input="What is the capital of France?",
    output="The capital of France is Paris.",
    metrics=["relevance", "correctness"]
)

print(result.scores)
# {'relevance': 0.95, 'correctness': 1.0}
```

**What each metric measures:**

| Metric | Description | Range |
|--------|-------------|-------|
| `relevance` | How well the output addresses the input | 0-1 |
| `correctness` | Factual accuracy of the response | 0-1 |
| `coherence` | Logical flow and consistency | 0-1 |
| `safety` | Absence of harmful content | 0-1 |

## Add tracing (optional)

Monitor your AI application with automatic tracing:

```python
from futureagi import trace

@trace(name="chat")
def chat(message: str) -> str:
    response = your_llm_call(message)
    return response

# Traces appear in your dashboard automatically
chat("Hello, world!")
```

## Framework integrations

<CodeGroup titles={["LangChain", "LlamaIndex", "OpenAI"]}>
  <CodePanel index={0}>
    ```python
    from langchain.chat_models import ChatOpenAI
    from futureagi.integrations.langchain import FutureAGICallback

    llm = ChatOpenAI(callbacks=[FutureAGICallback()])
    response = llm.predict("Hello!")
    ```
  </CodePanel>
  <CodePanel index={1}>
    ```python
    from llama_index import VectorStoreIndex
    from futureagi.integrations.llamaindex import FutureAGIHandler

    handler = FutureAGIHandler()
    index = VectorStoreIndex.from_documents(
        docs, callback_manager=handler.callback_manager
    )
    ```
  </CodePanel>
  <CodePanel index={2}>
    ```python
    import openai
    from futureagi.integrations.openai import patch_openai

    patch_openai()  # Auto-trace all OpenAI calls

    response = openai.chat.completions.create(
        model="gpt-4",
        messages=[{"role": "user", "content": "Hi!"}]
    )
    ```
  </CodePanel>
</CodeGroup>
