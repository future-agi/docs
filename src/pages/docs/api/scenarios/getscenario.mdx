# Get scenario details

Retrieves detailed information about a specific scenario, including graph data, prompts, and dataset row count.

<ApiPlayground
  method="GET"
  endpoint="/simulate/scenarios/{scenario_id}/"
  baseUrl="https://api.futureagi.com"
  parameters={[{"name": "scenario_id", "in": "path", "required": true, "description": "The UUID of the scenario.", "type": "string"}]}
/>

## Authentication

All requests require authentication via API keys:

| Header | Description |
|--------|-------------|
| `X-Api-Key` | Your API key |
| `X-Secret-Key` | Your secret key |

## Parameters

### Path Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `scenario_id` | string (UUID) | Yes | The UUID of the scenario to retrieve. |

## Responses

### 200

Scenario details returned successfully.

- **id**: string (UUID)
- **name**: string
- **description**: string
- **source**: string
- **scenario_type**: string
- **dataset_id**: string (UUID) or null
- **agent_type**: string
- **status**: string
- **graph**: object
  The graph structure associated with the scenario.
- **prompts**: array of objects
  Each object contains `role` and `content` fields.
- **dataset_rows**: integer
  Number of rows in the associated dataset.
- **created_at**: string (datetime)
- **updated_at**: string (datetime)

### 401

Unauthorized. Authentication credentials were not provided or are invalid.

### 404

Not Found. The scenario with the specified ID could not be found.

### 500

Internal Server Error. An unexpected error occurred.

## Code Examples

### cURL

```bash
curl -X GET "https://api.futureagi.com/simulate/scenarios/{scenario_id}/" \
  -H "X-Api-Key: YOUR_API_KEY" \
  -H "X-Secret-Key: YOUR_SECRET_KEY"
```

### Python

```python
import requests

url = "https://api.futureagi.com/simulate/scenarios/{scenario_id}/"
headers = {
    "X-Api-Key": "YOUR_API_KEY",
    "X-Secret-Key": "YOUR_SECRET_KEY"
}

response = requests.get(url, headers=headers)
print(response.json())
```

