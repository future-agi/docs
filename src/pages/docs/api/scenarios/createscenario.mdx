# Generate or create a scenario

Creates a new scenario from a dataset, a script, or a generated/provided graph. The creation is processed in the background.

/>

## Authentication

All requests require authentication via API keys:

| Header | Description |
|--------|-------------|
| `X-Api-Key` | Your API key |
| `X-Secret-Key` | Your secret key |

## Request Body

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `name` | string | Yes | Scenario name. Max 255 chars. Cannot be blank. |
| `description` | string | No | Optional description. |
| `kind` | string | No | Scenario type: `dataset`, `script`, or `graph`. Default: `dataset`. |
| `agentDefinitionId` | string (UUID) | Conditional | Agent definition ID. Required when `generateGraph` is `true` or `sourceType` is `agent_definition`. |
| `agentDefinitionVersionId` | string (UUID) | No | Specific agent version to use. |
| `datasetId` | string (UUID) | Conditional | Source dataset ID. Required if `kind` is `dataset`. |
| `scriptUrl` | string (URL) | Conditional | Script URL. Required if `kind` is `script`. |
| `noOfRows` | integer | No | Rows to generate. Range: 10-100. Default: `20`. |
| `addPersonaAutomatically` | boolean | No | Auto-add personas. Default: `false`. |
| `graph` | object | Conditional | Graph structure. Required if `kind` is `graph` and `generateGraph` is `false`. |
| `generateGraph` | boolean | No | Auto-generate graph (requires `agentDefinitionId`). Default: `false`. |
| `personas` | array of string | No | Persona UUIDs to include. |
| `sourceType` | string | No | Source: `agent_definition` or `prompt`. Default: `agent_definition`. |
| `promptTemplateId` | string (UUID) | Conditional | Prompt template ID. Required if `sourceType` is `prompt`. |
| `promptVersionId` | string (UUID) | Conditional | Prompt version ID. Required if `sourceType` is `prompt`. Must belong to the template. |
| `customColumns` | array of objects | No | Custom columns (max 10). Each needs `name` (max 50), `dataType`, `description` (max 200). No duplicates. |
| `customInstruction` | string | No | Custom instruction for the agent. |

### Example

```json
{
  "name": "your-name",
  "description": "your-description",
  "agentDefinitionId": "your-agentDefinitionId",
  "kind": "dataset",
  "datasetId": "your-datasetId",
  "noOfRows": 20,
  "addPersonaAutomatically": false,
  "sourceType": "agent_definition",
  "personas": [],
  "customColumns": [
    {
      "name": "expected_outcome",
      "dataType": "string",
      "description": "The expected outcome of the conversation"
    }
  ]
}
```

## Responses

### 202

Scenario creation has been accepted and is processing in the background.

- **message**: string
  Example: `Graph scenario creation started`
- **scenario**: any
- **status**: string
  Example: `processing`

### 400

Invalid data provided. Check for missing required fields based on the 'kind' of scenario.

- **error**: string
- **details**: object

### 401

Authentication credentials were not provided.

### 500

Internal server error occurred during scenario creation.

- **error**: string
  Example: `Failed to create scenario: [error details]`

## Code Examples

### cURL

```bash
curl -X POST "https://api.futureagi.com/simulate/scenarios/create/" \
  -H "X-Api-Key: YOUR_API_KEY" \
  -H "X-Secret-Key: YOUR_SECRET_KEY" \
  -H "Content-Type: application/json" \
  -d '{"name": "your-name", "description": "your-description", "agentDefinitionId": "your-agentDefinitionId", "kind": "dataset", "datasetId": "your-datasetId", "scriptUrl": "your-scriptUrl", "noOfRows": 1, "addPersonaAutomatically": true, "graph": {}, "generateGraph": true, "personas": []}'
```

### Python

```python
import requests

url = "https://api.futureagi.com/simulate/scenarios/create/"
headers = {
    "X-Api-Key": "YOUR_API_KEY",
    "X-Secret-Key": "YOUR_SECRET_KEY",
    "Content-Type": "application/json"
}
data = {
    "name": "your-name",
    "description": "your-description",
    "agentDefinitionId": "your-agentDefinitionId",
    "kind": "dataset",
    "datasetId": "your-datasetId",
    "scriptUrl": "your-scriptUrl",
    "noOfRows": 1,
    "addPersonaAutomatically": true,
    "graph": {},
    "generateGraph": true,
    "personas": []
}

response = requests.post(url, headers=headers, json=data)
print(response.json())
```
