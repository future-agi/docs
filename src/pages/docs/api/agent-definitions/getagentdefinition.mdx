# Get agent definition details

Retrieves detailed information about a specific agent definition, including all versions, the currently active version, and the total version count.

<ApiPlayground
  method="GET"
  endpoint="/simulate/agent-definitions/{agent_id}/"
  baseUrl="https://api.futureagi.com"
  parameters={[{"name": "agent_id", "in": "path", "required": true, "description": "A UUID string identifying the agent definition.", "type": "string"}]}
/>

## Authentication

All requests require authentication via API keys:

| Header | Description |
|--------|-------------|
| `X-Api-Key` | Your API key |
| `X-Secret-Key` | Your secret key |

## Parameters

### Path Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `agent_id` | string (UUID) | Yes | The UUID of the agent definition. |

## Responses

### 200

Agent definition details returned successfully.

- **id**: string (UUID)
- **agent_name**: string
- **agent_type**: string (`voice` or `text`)
- **contact_number**: string or null
- **inbound**: boolean
- **description**: string
- **assistant_id**: string or null
- **provider**: string or null
- **language**: string
- **languages**: array of string
- **websocket_url**: string or null
- **websocket_headers**: object or null
- **workspace**: string (UUID) or null
- **knowledge_base**: string (UUID) or null
- **organization**: string (UUID)
- **model**: string or null
- **model_details**: object or null
- **created_at**: string (datetime)
- **updated_at**: string (datetime)
- **versions**: array of objects
  List of all versions for this agent.
- **active_version**: object or null
  The currently active version.
- **version_count**: integer
  Total number of versions.

### 401

Unauthorized. Authentication credentials were not provided or are invalid.

### 404

Not Found. The agent definition with the specified ID could not be found.

### 500

Internal Server Error. An unexpected error occurred.
