# Fetch assistant from provider

Fetches assistant details from an external provider (VAPI, Retell, Eleven Labs) using the provided API key and assistant ID. Returns the assistant name and system prompt without creating a new agent version.

<ApiPlayground
  method="POST"
  endpoint="/simulate/api/agent-definition-operations/fetch_assistant_from_provider/"
  baseUrl="https://api.futureagi.com"
  parameters={[]}
  requestBody={{"assistantId": "your-assistant-id", "apiKey": "your-provider-api-key", "provider": "vapi"}}
/>

## Authentication

All requests require authentication via API keys:

| Header | Description |
|--------|-------------|
| `X-Api-Key` | Your API key |
| `X-Secret-Key` | Your secret key |

## Request Body

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `assistantId` | string | Yes | External assistant identifier from the provider. |
| `apiKey` | string | Yes | API key for authenticating with the provider. |
| `provider` | string | Yes | `vapi`, `retell`, `elevenLabs`, or `others`. |
| `name` | string | No | Name override. If omitted, fetched from provider. |
| `prompt` | string | No | Prompt override. If omitted, fetched from provider. |
| `commitMessage` | string | No | Auto-generated with timestamp if omitted. |

### Example

```json
{
  "assistantId": "your-assistant-id",
  "apiKey": "your-provider-api-key",
  "provider": "vapi"
}
```

## Responses

### 200

Assistant details fetched successfully.

- **assistantId**: string
- **apiKey**: string
- **name**: string
  Assistant name from the provider.
- **prompt**: string
  System prompt from the provider.
- **provider**: string
- **commitMessage**: string

### 400

Bad Request. Invalid API key or assistant ID.

- **error**: string
  Example: `Please recheck your API key and assistant ID`

### 401

Unauthorized. Authentication credentials were not provided or are invalid.

### 500

Internal Server Error. An unexpected error occurred.

## Code Examples

### cURL

```bash
curl -X POST "https://api.futureagi.com/simulate/api/agent-definition-operations/fetch_assistant_from_provider/" \
  -H "X-Api-Key: YOUR_API_KEY" \
  -H "X-Secret-Key: YOUR_SECRET_KEY" \
  -H "Content-Type: application/json" \
  -d '{"assistantId": "your-assistant-id", "apiKey": "your-provider-api-key", "provider": "vapi"}'
```

### Python

```python
import requests

url = "https://api.futureagi.com/simulate/api/agent-definition-operations/fetch_assistant_from_provider/"
headers = {
    "X-Api-Key": "YOUR_API_KEY",
    "X-Secret-Key": "YOUR_SECRET_KEY",
    "Content-Type": "application/json"
}
data = {
    "assistantId": "your-assistant-id",
    "apiKey": "your-provider-api-key",
    "provider": "vapi"
}

response = requests.post(url, headers=headers, json=data)
print(response.json())
```

### JavaScript
