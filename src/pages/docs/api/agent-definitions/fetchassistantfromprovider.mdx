---
title: "Fetch Assistant from Provider"
description: "Fetches assistant details (name and system prompt) from an external voice provider like VAPI, Retell, or Eleven Labs using the provided API key and assistant ID. Useful for previewing provider configuration before creating or updating an agent version."

<ApiSection title="Request body">
  <ParamField body="assistant_id" type="string" required>
    The external assistant identifier from the provider's platform. This is the ID of the assistant you want to fetch details for (e.g., your VAPI assistant ID or Retell agent ID).
  </ParamField>

  <ParamField body="api_key" type="string" required>
    The API key for authenticating with the external voice provider. This must be a valid key with permission to read assistant details from the provider's API.
  </ParamField>

  <ParamField body="provider" type="string" required>
    The voice provider to fetch from. Determines which provider API is called:
    - `"vapi"` — Fetches from the VAPI platform
    - `"retell"` — Fetches from the Retell AI platform
    - `"eleven_labs"` — Fetches from Eleven Labs
    - `"livekit"` — Fetches from the LiveKit platform
    - `"others"` — Custom provider
  </ParamField>

</ApiSection>
