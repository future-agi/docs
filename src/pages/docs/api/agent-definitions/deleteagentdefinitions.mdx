# Delete agent definitions

Soft-deletes one or more agent definitions and all their associated versions. The records are marked as deleted but not permanently removed.

<ApiPlayground
  method="DELETE"
  endpoint="/simulate/agent-definitions/"
  baseUrl="https://api.futureagi.com"
  parameters={[]}
  requestBody={{"agentIds": ["agent-uuid-1", "agent-uuid-2"]}}
/>

## Authentication

All requests require authentication via API keys:

| Header | Description |
|--------|-------------|
| `X-Api-Key` | Your API key |
| `X-Secret-Key` | Your secret key |

## Request Body

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `agentIds` | array of string (UUIDs) | Yes | One or more agent definition IDs to delete. |

### Validations

- `agentIds` must be a non-empty list.

### Example

```json
{
  "agentIds": ["a1b2c3d4-e5f6-7890-abcd-ef1234567890"]
}
```

## Responses

### 200

Agent definitions deleted successfully.

- **message**: string
  Example: `Agents deleted successfully`
- **agentsUpdated**: integer
  Number of agent definitions deleted.
- **versionsUpdated**: integer
  Number of agent versions deleted.

### 400

Bad Request. `agentIds` is missing or not a list.

### 401

Unauthorized. Authentication credentials were not provided or are invalid.

### 500

Internal Server Error. An unexpected error occurred.

## Code Examples

### cURL

```bash
curl -X DELETE "https://api.futureagi.com/simulate/agent-definitions/" \
  -H "X-Api-Key: YOUR_API_KEY" \
  -H "X-Secret-Key: YOUR_SECRET_KEY" \
  -H "Content-Type: application/json" \
  -d '{"agentIds": ["agent-uuid-1"]}'
```

### Python

```python
import requests

url = "https://api.futureagi.com/simulate/agent-definitions/"
headers = {
    "X-Api-Key": "YOUR_API_KEY",
    "X-Secret-Key": "YOUR_SECRET_KEY",
    "Content-Type": "application/json"
}
data = {
    "agentIds": ["agent-uuid-1"]
}

response = requests.delete(url, headers=headers, json=data)
print(response.json())
```

### JavaScript

