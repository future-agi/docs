# List agent definitions

Retrieves a paginated list of agent definitions for the authenticated user's organization, with optional search and filtering by agent type.

## Authentication

All requests require authentication via API keys:

| Header | Description |
|--------|-------------|
| `X-Api-Key` | Your API key |
| `X-Secret-Key` | Your secret key |

## Parameters

### Query Parameters

| Parameter | Type | Required | Default | Description |
|-----------|------|----------|---------|-------------|
| `search` | string | No | - | Case-insensitive search across agent name, contact number, description, and assistant ID. |
| `limit` | integer | No | 10 | Number of items per page. |
| `page` | integer | No | 1 | Page number. |
| `agent_type` | string | No | - | Filter by agent type. Values: `voice`, `text`. |
| `agent_definition_id` | string (UUID) | No | - | If provided, pins this agent as the first element on page 1. |

## Responses

### 200

A paginated list of agent definitions.

- **count**: integer
- **next**: string or null
- **previous**: string or null
- **results**: array of objects
  Each object contains:
  - **id**: string (UUID)
  - **agent_name**: string
  - **agent_type**: string (`voice` or `text`)
  - **contact_number**: string or null
  - **inbound**: boolean
  - **description**: string
  - **assistant_id**: string or null
  - **provider**: string or null
  - **language**: string
  - **languages**: array of string
  - **websocket_url**: string or null
  - **websocket_headers**: object or null
  - **workspace**: string (UUID) or null
  - **knowledge_base**: string (UUID) or null
  - **organization**: string (UUID)
  - **created_at**: string (datetime)
  - **updated_at**: string (datetime)
  - **latest_version**: integer
  - **latest_version_id**: string (UUID)
  - **model**: string or null
  - **model_details**: object or null

### 401

Unauthorized. Authentication credentials were not provided or are invalid.

### 500

Internal Server Error. An unexpected error occurred.

## Code Examples

### cURL

```bash
curl -X GET "https://api.futureagi.com/simulate/agent-definitions/?search=my-agent&limit=10&page=1" \
  -H "X-Api-Key: YOUR_API_KEY" \
  -H "X-Secret-Key: YOUR_SECRET_KEY"
```

### Python

