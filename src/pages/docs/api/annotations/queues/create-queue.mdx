# Create Queue

Create a new annotation queue to organize and distribute annotation work.

<ApiPlayground
  method="POST"
  endpoint="/model-hub/annotation-queues/"
  baseUrl="https://api.futureagi.com"
  parameters={[]}
  requestBody={{
    name: "My Annotation Queue",
    description: "Queue for reviewing trace outputs",
    instructions: "Rate each trace on accuracy and relevance",
    status: "draft",
    assignment_strategy: "round_robin",
    annotations_required: 2,
    reservation_timeout_minutes: 30,
    requires_review: false,
    label_ids: ["uuid-1", "uuid-2"],
    annotator_ids: ["uuid-a", "uuid-b"]
  }}
/>

## Authentication

| Header | Description |
|--------|-------------|
| `X-Api-Key` | Your API key |
| `X-Secret-Key` | Your secret key |

## Request Body

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `name` | string | Yes | Queue name |
| `description` | string | No | Queue description |
| `instructions` | string | No | Instructions for annotators |
| `status` | string | No | Initial status: `draft` (default), `active`, `paused` |
| `assignment_strategy` | string | No | `manual`, `round_robin` (default), or `load_balanced` |
| `annotations_required` | integer | No | Number of annotations required per item (default: 1) |
| `reservation_timeout_minutes` | integer | No | Minutes before a reserved item is released (default: 30) |
| `requires_review` | boolean | No | Whether completed items require review (default: false) |
| `label_ids` | array | No | List of annotation label UUIDs to attach |
| `annotator_ids` | array | No | List of user UUIDs to assign as annotators |

## Response

Returns the created queue object with all fields including `id`, `created_at`, and `updated_at`.
