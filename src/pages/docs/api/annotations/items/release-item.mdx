# Release Item

Release a queue item that is currently reserved or in progress, making it available for reassignment.

<ApiPlayground
  method="POST"
  endpoint="/model-hub/annotation-queues/{queue_id}/items/{item_id}/release/"
  baseUrl="https://api.futureagi.com"
  parameters={[
    { name: "queue_id", type: "string", description: "Queue UUID", required: true },
    { name: "item_id", type: "string", description: "Item UUID", required: true }
  ]}
/>

## Authentication

| Header | Description |
|--------|-------------|
| `X-Api-Key` | Your API key |
| `X-Secret-Key` | Your secret key |

## Path Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `queue_id` | UUID | Yes | The annotation queue ID |
| `item_id` | UUID | Yes | The queue item ID |

## Response

Returns the updated item object with assignment cleared and status set back to `pending`.
