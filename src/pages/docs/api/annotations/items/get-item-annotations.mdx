# Get Item Annotations

Retrieve all annotations and notes that have been submitted for a queue item.

<ApiPlayground
  method="GET"
  endpoint="/model-hub/annotation-queues/{queue_id}/items/{item_id}/annotations/"
  baseUrl="https://api.futureagi.com"
  parameters={[
    { name: "queue_id", type: "string", description: "Queue UUID", required: true },
    { name: "item_id", type: "string", description: "Item UUID", required: true }
  ]}
/>

## Authentication

| Header | Description |
|--------|-------------|
| `X-Api-Key` | Your API key |
| `X-Secret-Key` | Your secret key |

## Path Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `queue_id` | UUID | Yes | The annotation queue ID |
| `item_id` | UUID | Yes | The queue item ID |

## Response

Returns an array of annotation objects, each containing `id`, `label_id`, `label_name`, `value`, `annotator`, `created_at`, and any associated notes.

## Code Examples

<CodeGroup>
