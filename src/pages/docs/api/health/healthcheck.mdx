# Health check

Returns 200 status when server is up and running. No authentication required.

<ApiPlayground
  method="GET"
  endpoint="/health/"
  baseUrl="https://api.futureagi.com"
  parameters={[]}
  
/>

## Responses

### 200

Server is healthy and running

```json
{
  "status": true,
  "result": "Server is up and running"
}
```

### 500

Internal server error

- **error**: string
  Example: `Internal server error`

## Code Examples

### cURL

```bash
curl -X GET "https://api.futureagi.com/health/" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json"
```

### Python

```python
import requests

url = "https://api.futureagi.com/health/"
headers = {
    "Authorization": "Bearer YOUR_API_KEY",
    "Content-Type": "application/json"
}

response = requests.get(url, headers=headers)
print(response.json())
```

### JavaScript

```javascript
const response = await fetch("https://api.futureagi.com/health/", {
  method: "GET",
  headers: {
    "Authorization": "Bearer YOUR_API_KEY",
    "Content-Type": "application/json"
  },
});

const data = await response.json();
console.log(data);
```
