# Update Custom Eval Config

Updates an existing custom eval config. Use PUT for full replacement or PATCH for partial updates.

<ApiPlayground
  method="PATCH"
  endpoint="/tracer/custom-eval-config/{id}/"
  baseUrl="https://api.futureagi.com"
  parameters={[{"name": "id", "in": "path", "required": true, "description": "UUID of the custom eval config to update.", "type": "string"}]}
  requestBody={{
    name: "updated-config-name"
  }}
/>

## Authentication

All requests require authentication via API keys:

| Header | Description |
|--------|-------------|
| `X-Api-Key` | Your API key |
| `X-Secret-Key` | Your secret key |

## Parameters

<ParamField body="id" type="string" required>
  UUID of the custom eval config to update. Passed as a path parameter.
</ParamField>

## Request Body

All fields are optional for PATCH. For PUT, all required create fields apply.

<ParamField body="eval_template" type="string">
  UUID of the eval template.
</ParamField>

<ParamField body="name" type="string">
  Config name. Max 255 characters. Must be unique per project among non-deleted configs.
</ParamField>

<ParamField body="config" type="object">
  Configuration dictionary. Normalized against the eval template's config schema.
</ParamField>

<ParamField body="mapping" type="object">
  Field mapping configuration.
</ParamField>

<ParamField body="filters" type="object">
  Filter criteria.
</ParamField>

<ParamField body="error_localizer" type="boolean">
  Enable or disable error localization.
</ParamField>

<ParamField body="kb_id" type="string">
  UUID of a knowledge base file to reference.
</ParamField>

<ParamField body="model" type="string">
  Model to use. One of: `turing_large`, `turing_small`, `protect`, `protect_flash`, `turing_flash`.
</ParamField>

### Responses

| Status | Description |
|--------|-------------|
| `200` | Custom eval config updated successfully. |
| `400` | Invalid data or duplicate name within the project. |
| `401` | Unauthorized. Authentication credentials were not provided or are invalid. |
| `404` | Custom eval config not found. |
| `500` | Internal Server Error. |

## Code Examples

<CodeGroup>
```python Python
import requests

url = "https://api.futureagi.com/tracer/custom-eval-config/{id}/"
headers = {
    "X-Api-Key": "YOUR_API_KEY",
    "X-Secret-Key": "YOUR_SECRET_KEY",
    "Content-Type": "application/json",
}

payload = {
    "name": "updated-config-name",
}

response = requests.patch(url, json=payload, headers=headers)
print(response.json())
```
