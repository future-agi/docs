# List Custom Eval Configs

Retrieves a list of all custom eval configs for the authenticated user's organization. Supports filtering by eval template, name, or project.

<ApiPlayground
  method="GET"
  endpoint="/tracer/custom-eval-config/"
  baseUrl="https://api.futureagi.com"
  parameters={[{"name": "eval_template_id", "in": "query", "required": false, "description": "Filter by eval template UUID.", "type": "string"}, {"name": "name", "in": "query", "required": false, "description": "Filter by config name (case-insensitive).", "type": "string"}, {"name": "project_id", "in": "query", "required": false, "description": "Filter by project UUID.", "type": "string"}]}
/>

## Authentication

All requests require authentication via API keys:

| Header | Description |
|--------|-------------|
| `X-Api-Key` | Your API key |
| `X-Secret-Key` | Your secret key |

## Parameters

### Query Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `eval_template_id` | string (UUID) | No | Filter by eval template. |
| `name` | string | No | Filter by config name (case-insensitive search). |
| `project_id` | string (UUID) | No | Filter by project. |

## Responses

### 200

Successfully retrieved the list of custom eval configs.

- **status**: string
- **result**: array
  Array of custom eval config objects, each containing `id`, `eval_template`, `name`, `config`, `mapping`, `project`, `filters`, `error_localizer`, `kb_id`, `model`, `eval_group`.

### 401

Unauthorized. Authentication credentials were not provided or are invalid.

### 500

Internal Server Error. An unexpected error occurred.

### cURL

```bash

  -H "X-Api-Key: YOUR_API_KEY" \
  -H "X-Secret-Key: YOUR_SECRET_KEY" \
  -H "Content-Type: application/json"
```
