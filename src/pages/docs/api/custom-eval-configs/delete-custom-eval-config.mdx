# Delete Custom Eval Config

Soft deletes a custom eval config by marking it as deleted with a timestamp. The config name becomes available for reuse after deletion.

<ApiPlayground
  method="DELETE"
  endpoint="/tracer/custom-eval-config/{id}/"
  baseUrl="https://api.futureagi.com"
  parameters={[{"name": "id", "in": "path", "required": true, "description": "UUID of the custom eval config to delete.", "type": "string"}]}
/>

## Authentication

All requests require authentication via API keys:

| Header | Description |
|--------|-------------|
| `X-Api-Key` | Your API key |
| `X-Secret-Key` | Your secret key |

## Parameters

<ParamField body="id" type="string" required>
  UUID of the custom eval config to delete. Passed as a path parameter.
</ParamField>

### Responses

| Status | Description |
|--------|-------------|
| `204` | Custom eval config deleted successfully. |
| `401` | Unauthorized. Authentication credentials were not provided or are invalid. |
| `404` | Custom eval config not found. |

## Code Examples

<CodeGroup>
```python Python
import requests

url = "https://api.futureagi.com/tracer/custom-eval-config/{id}/"
headers = {
    "X-Api-Key": "YOUR_API_KEY",
    "X-Secret-Key": "YOUR_SECRET_KEY",
}

response = requests.delete(url, headers=headers)
print(response.status_code)
```
