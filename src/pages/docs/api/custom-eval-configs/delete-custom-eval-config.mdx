# Delete Custom Eval Config

Soft deletes a custom eval config by marking it as deleted with a timestamp. The config name becomes available for reuse after deletion.

<ApiPlayground
  method="DELETE"
  endpoint="/tracer/custom-eval-config/{id}/"
  baseUrl="https://api.futureagi.com"
  parameters={[{"name": "id", "in": "path", "required": true, "description": "The UUID of the custom eval config to delete.", "type": "string"}]}
/>

## Authentication

All requests require authentication via API keys:

| Header | Description |
|--------|-------------|
| `X-Api-Key` | Your API key |
| `X-Secret-Key` | Your secret key |

## Parameters

### Path Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `id` | string (UUID) | Yes | The UUID of the custom eval config to delete. |

## Responses

### 204

Custom eval config deleted successfully.

### 401

Unauthorized. Authentication credentials were not provided or are invalid.

### 404

Not Found. The specified custom eval config does not exist.

## Code Examples

### cURL

```bash
curl -X DELETE "https://api.futureagi.com/tracer/custom-eval-config/{id}/" \
  -H "X-Api-Key: YOUR_API_KEY" \
  -H "X-Secret-Key: YOUR_SECRET_KEY" \
  -H "Content-Type: application/json"
```

### Python

```python
import requests

url = "https://api.futureagi.com/tracer/custom-eval-config/{id}/"
headers = {
    "X-Api-Key": "YOUR_API_KEY",
    "X-Secret-Key": "YOUR_SECRET_KEY",
    "Content-Type": "application/json"
}

response = requests.delete(url, headers=headers)
print(response.status_code)
```

### JavaScript

```javascript

```
