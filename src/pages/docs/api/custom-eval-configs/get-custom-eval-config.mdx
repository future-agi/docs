# Get Custom Eval Config

Retrieves detailed information about a specific custom eval config by its ID.

<ApiPlayground
  method="GET"
  endpoint="/tracer/custom-eval-config/{id}/"
  baseUrl="https://api.futureagi.com"
  parameters={[{"name": "id", "in": "path", "required": true, "description": "The UUID of the custom eval config to retrieve.", "type": "string"}]}
/>

## Authentication

All requests require authentication via API keys:

| Header | Description |
|--------|-------------|
| `X-Api-Key` | Your API key |
| `X-Secret-Key` | Your secret key |

## Parameters

### Path Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `id` | string (UUID) | Yes | The UUID of the custom eval config to retrieve. |

## Responses

### 200

Successfully retrieved the custom eval config.

- **status**: string
- **result**: object
  Custom eval config object containing `id`, `eval_template`, `name`, `config`, `mapping`, `project`, `filters`, `error_localizer`, `kb_id`, `model`, `eval_group`.

### 401

Unauthorized. Authentication credentials were not provided or are invalid.

### 404

Not Found. The specified custom eval config does not exist.

## Code Examples

### cURL

```bash
curl -X GET "https://api.futureagi.com/tracer/custom-eval-config/{id}/" \
  -H "X-Api-Key: YOUR_API_KEY" \
  -H "X-Secret-Key: YOUR_SECRET_KEY" \
  -H "Content-Type: application/json"
```

### Python

```python
import requests

url = "https://api.futureagi.com/tracer/custom-eval-config/{id}/"
headers = {
    "X-Api-Key": "YOUR_API_KEY",
    "X-Secret-Key": "YOUR_SECRET_KEY",
    "Content-Type": "application/json"
}

response = requests.get(url, headers=headers)
print(response.json())
```
