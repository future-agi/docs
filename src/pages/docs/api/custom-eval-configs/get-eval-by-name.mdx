# Get Eval Template by Name

Retrieves a user-owned eval template by its exact name. Returns the template details if found, or indicates that no user template exists with the given name.

<ApiPlayground
  method="POST"
  endpoint="/tracer/custom-eval-config/get_custom_eval_by_name/"
  baseUrl="https://api.futureagi.com"
  parameters={[]}
  requestBody={{"eval_template_name": "my-custom-eval"}}
/>

## Authentication

All requests require authentication via API keys:

| Header | Description |
|--------|-------------|
| `X-Api-Key` | Your API key |
| `X-Secret-Key` | Your secret key |

## Request Body

- **eval_template_name** (required): string
  The exact name of the eval template to look up. Searches for templates owned by the user within their organization.

### Example

```json
{
  "eval_template_name": "my-custom-eval"
}
```

## Responses

### 200

Lookup completed successfully.

- **status**: string
- **result**: object
  - **is_user_eval_template**: boolean — `true` if a matching user-owned template was found.
  - **eval_template**: object or null — The eval template details if found, otherwise `null`.

### 400

Bad Request. Missing required field `eval_template_name`.

### 401

Unauthorized. Authentication credentials were not provided or are invalid.

### 500

Internal Server Error. An unexpected error occurred.

## Code Examples

### cURL

```bash
curl -X POST "https://api.futureagi.com/tracer/custom-eval-config/get_custom_eval_by_name/" \
  -H "X-Api-Key: YOUR_API_KEY" \
  -H "X-Secret-Key: YOUR_SECRET_KEY" \
  -H "Content-Type: application/json" \
  -d '{"eval_template_name": "my-custom-eval"}'
```

### Python

```python
import requests

url = "https://api.futureagi.com/tracer/custom-eval-config/get_custom_eval_by_name/"
headers = {
    "X-Api-Key": "YOUR_API_KEY",
    "X-Secret-Key": "YOUR_SECRET_KEY",
    "Content-Type": "application/json"
}
data = {
    "eval_template_name": "my-custom-eval"
}

response = requests.post(url, headers=headers, json=data)
print(response.json())
```
