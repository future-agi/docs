# Check Config Exists

Checks if a custom eval config with the same name and mapping already exists in a project. Useful for validating eval tags before creating configs programmatically.

<ApiPlayground
  method="POST"
  endpoint="/tracer/custom-eval-config/check_exists/"
  baseUrl="https://api.futureagi.com"
  parameters={[]}
  requestBody={{
    project_name: "my-project",
    eval_tags: [{ custom_eval_name: "my-eval", eval_name: "relevance", mapping: {} }]
  }}
/>

## Authentication

All requests require authentication via API keys:

| Header | Description |
|--------|-------------|
| `X-Api-Key` | Your API key |
| `X-Secret-Key` | Your secret key |

## Request Body

<ParamField body="project_name" type="string" required>
  Name of the project to check against.
</ParamField>

<ParamField body="project_type" type="string" default="experiment">
  Type of the project.
</ParamField>

<ParamField body="eval_tags" type="array" required>
  List of eval tag objects to check. Each object must contain:
  - `custom_eval_name` (string) — The custom eval config name.
  - `eval_name` (string) — The eval template name.
  - `mapping` (object) — The field mapping to compare.
</ParamField>

## Response

<ResponseField name="exists" type="boolean">`true` if a conflicting config was found, `false` if all configs are valid.</ResponseField>
<ResponseField name="message" type="string">Descriptive message explaining the result.</ResponseField>

Possible messages:
- `"Project {name} does not exist"` — Project not found.
- `"Custom eval '{name}' already exists in project '{project}' with different configuration or mapping"` — Conflict detected.
- `"All custom eval configurations are valid"` — No conflicts.

### Responses

| Status | Description |
|--------|-------------|
| `200` | Check completed successfully. |
| `400` | Missing required fields (`project_name`, `eval_tags`), `eval_tags` is not a list, or invalid JSON in mapping. |
| `401` | Unauthorized. Authentication credentials were not provided or are invalid. |
| `500` | Internal Server Error. |
