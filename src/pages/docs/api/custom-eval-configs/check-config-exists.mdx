# Check Config Exists

Checks if a custom eval config with the same name and mapping already exists in a project. Useful for validating eval tags before creating configs programmatically.

<ApiPlayground
  method="POST"
  endpoint="/tracer/custom-eval-config/check_exists/"
  baseUrl="https://api.futureagi.com"
  parameters={[]}
  requestBody={{"project_name": "my-project", "eval_tags": [{"custom_eval_name": "my-eval", "eval_name": "relevance", "mapping": {}}]}}
/>

## Authentication

All requests require authentication via API keys:

| Header | Description |
|--------|-------------|
| `X-Api-Key` | Your API key |
| `X-Secret-Key` | Your secret key |

## Request Body

- **project_name** (required): string
  Name of the project to check against.
- **project_type**: string
  Type of the project. Default: `"experiment"`.
- **eval_tags** (required): array of object
  List of eval tag objects to check. Each object must contain:
  - **custom_eval_name**: string — The custom eval config name.
  - **eval_name**: string — The eval template name.
  - **mapping**: object — The field mapping to compare.

### Example

```json
{
  "project_name": "my-project",
  "eval_tags": [
    {
      "custom_eval_name": "my-relevance-eval",
      "eval_name": "relevance",
      "mapping": {
        "input": "user_query",
        "output": "assistant_response"
      }
    }
  ]
}
```

## Responses

### 200

Check completed successfully.

- **status**: string
- **result**: object
  - **exists**: boolean — `true` if a conflicting config was found, `false` if all configs are valid.
  - **message**: string — Descriptive message. Possible values:
    - `"Project {name} does not exist"` — Project not found.
    - `"Custom eval '{name}' already exists in project '{project}' with different configuration or mapping"` — Conflict detected.
    - `"All custom eval configurations are valid"` — No conflicts.

### 400

Bad Request. Possible reasons:
- Missing required fields (`project_name`, `eval_tags`).
- `eval_tags` is not a list.
- Invalid JSON in mapping fields.

### 401

Unauthorized. Authentication credentials were not provided or are invalid.

### 500

Internal Server Error. An unexpected error occurred.
