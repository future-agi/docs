title: "Delete a test run"
description: "Soft-deletes a test run. Cannot delete a test run that has running executions."
---

<ApiSection title="Path parameters">
  <ParamField path="run_test_id" type="string" required>
    The UUID of the test run to delete. This identifier is returned when you create a test run via the [Create Test Run](/docs/api/run-tests/createruntest) endpoint, or can be found in the results from the [List Test Runs](/docs/api/run-tests/listruntests) endpoint. The test run must not have any currently running executions -- if it does, you must wait for them to complete or cancel them before deleting.
  </ParamField>
</ApiSection>

<ApiSection title="Response" status={200} statusText="OK">
  <ResponseField name="message" type="string">
    A confirmation message indicating the test run was successfully soft-deleted. The value is `"Run test deleted successfully"`. Note that this is a soft delete -- the record is marked as deleted but not permanently removed from the database, allowing for potential recovery by Future AGI support if needed.
  </ResponseField>
</ApiSection>
