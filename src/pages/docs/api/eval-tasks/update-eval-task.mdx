# Update Eval Task

Updates an existing eval task's details. Supports full replacement via PUT or partial updates via PATCH.

<ApiPlayground

  endpoint="/tracer/eval-task/{id}/"
  baseUrl="https://api.futureagi.com"

  requestBody={{"name": "updated-task-name", "sampling_rate": 50.0}}
/>

## Authentication

All requests require authentication via API keys:

| Header | Description |
|--------|-------------|
| `X-Api-Key` | Your API key |
| `X-Secret-Key` | Your secret key |

## Parameters

### Path Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `id` | string (UUID) | Yes | The UUID of the eval task to update. |

## Request Body

All fields are optional for PATCH. For PUT, all required fields from create apply.

- **name**: string
  Task name. Min length: 1, max length: 255 characters.
- **evals**: array of string (UUID)
  List of custom eval config IDs. Must contain at least one valid, non-deleted config.
- **sampling_rate**: number
  Percentage of spans to evaluate. Min: 1.0, max: 100.0.
- **run_type**: string
  Must be `"continuous"` or `"historical"`.
- **spans_limit**: integer
  Max spans to evaluate. Required when `run_type` is `"historical"`. Min: 1, max: 1,000,000.
- **filters**: object
  Query filters to scope the spans.

## Responses

### 200

Eval task updated successfully.

### 400

Bad Request. Validation failed or task is in a state that cannot be updated (e.g., running — must pause first).

### 401

Unauthorized. Authentication credentials were not provided or are invalid.

### 404

Not Found. The specified eval task does not exist.

## Code Examples

### cURL

```bash
curl -X PATCH "https://api.futureagi.com/tracer/eval-task/{id}/" \
  -H "X-Api-Key: YOUR_API_KEY" \
  -H "X-Secret-Key: YOUR_SECRET_KEY" \
  -H "Content-Type: application/json" \
  -d '{"name": "updated-task-name", "sampling_rate": 50.0}'
```

### Python

```python
import requests

url = "https://api.futureagi.com/tracer/eval-task/{id}/"
headers = {
    "X-Api-Key": "YOUR_API_KEY",
    "X-Secret-Key": "YOUR_SECRET_KEY",
    "Content-Type": "application/json"
}
data = {
    "name": "updated-task-name",
    "sampling_rate": 50.0
}

response = requests.patch(url, headers=headers, json=data)
print(response.json())
```

### JavaScript

