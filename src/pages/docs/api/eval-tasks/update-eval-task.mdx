# Update Eval Task

Updates an existing eval task. Use PUT for full replacement or PATCH for partial updates.

<ApiPlayground
  method="PATCH"
  endpoint="/tracer/eval-task/{id}/"
  baseUrl="https://api.futureagi.com"
  parameters={[{"name": "id", "in": "path", "required": true, "description": "UUID of the eval task to update.", "type": "string"}]}
  requestBody={{
    name: "updated-task-name",
    sampling_rate: 50.0
  }}
/>

## Authentication

All requests require authentication via API keys:

| Header | Description |
|--------|-------------|
| `X-Api-Key` | Your API key |
| `X-Secret-Key` | Your secret key |

## Parameters

<ParamField body="id" type="string" required>
  UUID of the eval task to update. Passed as a path parameter.
</ParamField>

## Request Body

All fields are optional for PATCH. For PUT, all required create fields apply.

<ParamField body="name" type="string">
  Task name. Must be between 1 and 255 characters.
</ParamField>

<ParamField body="evals" type="array">
  List of custom eval config UUIDs. Must contain at least one valid, non-deleted config.
</ParamField>

<ParamField body="sampling_rate" type="number">
  Percentage of spans to evaluate. Must be between `1.0` and `100.0`.
</ParamField>

<ParamField body="run_type" type="string">
  Execution mode. Must be `"continuous"` or `"historical"`.
</ParamField>

<ParamField body="spans_limit" type="integer">
  Maximum spans to evaluate. Required when `run_type` is `"historical"`. Must be between `1` and `1000000`.
</ParamField>

<ParamField body="filters" type="object">
  Query filters to scope which spans are evaluated.
</ParamField>

<Note>
  Running tasks cannot be updated directly — they must be paused first.
</Note>

### Responses

| Status | Description |
|--------|-------------|
| `200` | Eval task updated successfully. |
| `400` | Validation failed or task is in a state that cannot be updated (e.g., running). |
| `401` | Unauthorized. Authentication credentials were not provided or are invalid. |
| `404` | Eval task not found. |

## Code Examples

<CodeGroup>
```python Python
import requests

url = "https://api.futureagi.com/tracer/eval-task/{id}/"
headers = {
    "X-Api-Key": "YOUR_API_KEY",
    "X-Secret-Key": "YOUR_SECRET_KEY",
    "Content-Type": "application/json",
}

payload = {
    "name": "updated-task-name",
    "sampling_rate": 50.0,
}

response = requests.patch(url, json=payload, headers=headers)
print(response.json())
```
