# Pause Eval Task

Pauses a currently running eval task. Only tasks with status `"running"` can be paused.

<ApiPlayground
  method="POST"
  endpoint="/tracer/eval-task/pause_eval_task/"
  baseUrl="https://api.futureagi.com"
  parameters={[{"name": "eval_task_id", "in": "query", "required": true, "description": "The UUID of the eval task to pause.", "type": "string"}]}
/>

## Authentication

All requests require authentication via API keys:

| Header | Description |
|--------|-------------|
| `X-Api-Key` | Your API key |
| `X-Secret-Key` | Your secret key |

## Parameters

### Query Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `eval_task_id` | string (UUID) | Yes | The UUID of the eval task to pause. |

## Responses

### 200

Eval task paused successfully.

- **status**: boolean
- **result**: string — Confirmation message.

### 400

Bad Request. The task is not in a `"running"` state and cannot be paused.

### 401

Unauthorized. Authentication credentials were not provided or are invalid.

### 404

Not Found. The specified eval task does not exist.

## Code Examples

### cURL

```bash
curl -X POST "https://api.futureagi.com/tracer/eval-task/pause_eval_task/?eval_task_id=YOUR_EVAL_TASK_ID" \
  -H "X-Api-Key: YOUR_API_KEY" \
  -H "X-Secret-Key: YOUR_SECRET_KEY" \
  -H "Content-Type: application/json"
```

### Python

```python
import requests

url = "https://api.futureagi.com/tracer/eval-task/pause_eval_task/"
headers = {
    "X-Api-Key": "YOUR_API_KEY",
    "X-Secret-Key": "YOUR_SECRET_KEY",
    "Content-Type": "application/json"
}
params = {
    "eval_task_id": "a1b2c3d4-e5f6-7890-abcd-ef1234567890"
}

response = requests.post(url, headers=headers, params=params)
print(response.json())
```
