# Pause Eval Task

Pauses a currently running eval task. Only tasks with status `"running"` can be paused.

<ApiPlayground
  method="POST"
  endpoint="/tracer/eval-task/pause_eval_task/"
  baseUrl="https://api.futureagi.com"
  parameters={[{"name": "eval_task_id", "in": "query", "required": true, "description": "UUID of the eval task to pause.", "type": "string"}]}
/>

## Authentication

All requests require authentication via API keys:

| Header | Description |
|--------|-------------|
| `X-Api-Key` | Your API key |
| `X-Secret-Key` | Your secret key |

## Query Parameters

<ParamField body="eval_task_id" type="string" required>
  UUID of the eval task to pause.
</ParamField>

## Response

<ResponseField name="status" type="boolean">Indicates success.</ResponseField>
<ResponseField name="result" type="string">Confirmation message.</ResponseField>

### Responses

| Status | Description |
|--------|-------------|
| `200` | Eval task paused successfully. |
| `400` | Task is not in `"running"` state and cannot be paused. |
| `401` | Unauthorized. Authentication credentials were not provided or are invalid. |
| `404` | Eval task not found. |
