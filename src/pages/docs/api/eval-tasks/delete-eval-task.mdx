# Delete Eval Task

Soft deletes an eval task and its associated logger records. Running tasks cannot be deleted — they must be paused first.

<ApiPlayground
  method="DELETE"
  endpoint="/tracer/eval-task/{id}/"
  baseUrl="https://api.futureagi.com"
  parameters={[{"name": "id", "in": "path", "required": true, "description": "The UUID of the eval task to delete.", "type": "string"}]}
/>

## Authentication

All requests require authentication via API keys:

| Header | Description |
|--------|-------------|
| `X-Api-Key` | Your API key |
| `X-Secret-Key` | Your secret key |

## Parameters

### Path Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `id` | string (UUID) | Yes | The UUID of the eval task to delete. |

## Responses

### 204

Eval task deleted successfully.

### 400

Bad Request. The task is currently running and must be paused before deletion.

### 401

Unauthorized. Authentication credentials were not provided or are invalid.

### 404

Not Found. The specified eval task does not exist.

## Code Examples

### cURL

```bash
curl -X DELETE "https://api.futureagi.com/tracer/eval-task/{id}/" \
  -H "X-Api-Key: YOUR_API_KEY" \
  -H "X-Secret-Key: YOUR_SECRET_KEY" \
  -H "Content-Type: application/json"
```

### Python

```python
import requests

url = "https://api.futureagi.com/tracer/eval-task/{id}/"
headers = {
    "X-Api-Key": "YOUR_API_KEY",
    "X-Secret-Key": "YOUR_SECRET_KEY",
    "Content-Type": "application/json"
}

response = requests.delete(url, headers=headers)
print(response.status_code)
```
