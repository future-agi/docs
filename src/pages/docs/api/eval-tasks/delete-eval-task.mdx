# Delete Eval Task

Soft deletes an eval task and its associated logger records.

<ApiPlayground
  method="DELETE"
  endpoint="/tracer/eval-task/{id}/"
  baseUrl="https://api.futureagi.com"
  parameters={[{"name": "id", "in": "path", "required": true, "description": "UUID of the eval task to delete.", "type": "string"}]}
/>

## Authentication

All requests require authentication via API keys:

| Header | Description |
|--------|-------------|
| `X-Api-Key` | Your API key |
| `X-Secret-Key` | Your secret key |

## Parameters

<ParamField body="id" type="string" required>
  UUID of the eval task to delete. Passed as a path parameter.
</ParamField>

<Note>
  Running tasks cannot be deleted — they must be paused first.
</Note>

### Responses

| Status | Description |
|--------|-------------|
| `204` | Eval task deleted successfully. |
| `400` | Task is currently running and must be paused before deletion. |
| `401` | Unauthorized. Authentication credentials were not provided or are invalid. |
| `404` | Eval task not found. |
