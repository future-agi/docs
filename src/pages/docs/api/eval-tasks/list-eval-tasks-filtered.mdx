# List Eval Tasks

Retrieves a paginated list of eval tasks, with optional filtering by project or name.

<ApiPlayground
  method="GET"
  endpoint="/tracer/eval-task/list_eval_tasks/"
  baseUrl="https://api.futureagi.com"
  parameters={[
    {"name": "project_id", "in": "query", "required": false, "description": "Filter by project UUID.", "type": "string"},
    {"name": "name", "in": "query", "required": false, "description": "Search by task name (case-insensitive partial match).", "type": "string"},
    {"name": "page_number", "in": "query", "required": false, "description": "Page number (default: 0).", "type": "integer"},
    {"name": "page_size", "in": "query", "required": false, "description": "Results per page (default: 30).", "type": "integer"}
  ]}
/>

## Authentication

All requests require authentication via API keys:

| Header | Description |
|--------|-------------|
| `X-Api-Key` | Your API key |
| `X-Secret-Key` | Your secret key |

## Query Parameters

<ParamField body="project_id" type="string">
  Filter eval tasks by project UUID. Also accepts `projectId`.
</ParamField>

<ParamField body="name" type="string">
  Search by task name. Performs a case-insensitive partial match.
</ParamField>

<ParamField body="page_number" type="integer" default="0">
  Page number for pagination.
</ParamField>

<ParamField body="page_size" type="integer" default="30">
  Number of results per page.
</ParamField>

## Response

<ResponseField name="metadata" type="object">Contains `total_rows` — the total number of matching eval tasks.</ResponseField>
<ResponseField name="table" type="array">Array of eval task objects with `id`, `name`, `status`, `filters_applied`, `created_at`, `evals_applied`, `sampling_rate`, `last_run`.</ResponseField>
<ResponseField name="config" type="object">Table column configuration.</ResponseField>

### Responses

| Status | Description |
|--------|-------------|
| `200` | Successfully retrieved the list. |
| `401` | Unauthorized. Authentication credentials were not provided or are invalid. |
| `500` | Internal Server Error. |
