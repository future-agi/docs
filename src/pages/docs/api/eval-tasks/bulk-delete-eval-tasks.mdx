---
title: "Bulk Delete Eval Tasks"
description: "Soft deletes multiple eval tasks and their associated evaluation logger records in a single request. This is useful for cleaning up completed or obsolete tasks in bulk rather than deleting them one at a time."
---

<ApiSection title="Request body">
  <ParamField body="eval_task_ids" type="array" required>
    A list of eval task UUIDs to mark as deleted. All tasks in the list must be in a non-running state — the entire request will fail if any task is currently `running`. Use the [Pause Eval Task](/docs/api/eval-tasks/pause-eval-task) endpoint to pause active tasks before including them in a bulk delete.
  </ParamField>
</ApiSection>

<ApiSection title="Response" status={200} statusText="OK">

  </ResponseField>
  <ResponseField name="result" type="string">
    A human-readable confirmation message describing the outcome of the operation.
  </ResponseField>
</ApiSection>
