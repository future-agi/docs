# Bulk Delete Eval Tasks

Soft deletes multiple eval tasks and their associated logger and eval logger records in a single request. Running tasks cannot be deleted — they must be paused first.

<ApiPlayground
  method="POST"
  endpoint="/tracer/eval-task/mark_eval_tasks_deleted/"
  baseUrl="https://api.futureagi.com"
  parameters={[]}
  requestBody={{"eval_task_ids": ["task-uuid-1", "task-uuid-2"]}}
/>

## Authentication

All requests require authentication via API keys:

| Header | Description |
|--------|-------------|
| `X-Api-Key` | Your API key |
| `X-Secret-Key` | Your secret key |

## Request Body

- **eval_task_ids** (required): array of string (UUID)
  List of eval task UUIDs to mark as deleted.

### Example

```json
{
  "eval_task_ids": [
    "a1b2c3d4-e5f6-7890-abcd-ef1234567890",
    "d4e5f6a7-b8c9-0123-def0-123456789abc"
  ]
}
```

## Responses

### 200

Eval tasks deleted successfully.

- **status**: boolean
- **result**: string — Confirmation message.

### 400

Bad Request. Possible reasons:
- One or more tasks are currently running and must be paused before deletion.
- One or more task IDs do not exist.

### 401

Unauthorized. Authentication credentials were not provided or are invalid.

## Code Examples

### cURL

```bash
curl -X POST "https://api.futureagi.com/tracer/eval-task/mark_eval_tasks_deleted/" \
  -H "X-Api-Key: YOUR_API_KEY" \
  -H "X-Secret-Key: YOUR_SECRET_KEY" \
  -H "Content-Type: application/json" \
  -d '{"eval_task_ids": ["a1b2c3d4-e5f6-7890-abcd-ef1234567890"]}'
```

### Python

```python
import requests

url = "https://api.futureagi.com/tracer/eval-task/mark_eval_tasks_deleted/"
headers = {
    "X-Api-Key": "YOUR_API_KEY",
    "X-Secret-Key": "YOUR_SECRET_KEY",
    "Content-Type": "application/json"
}
data = {
    "eval_task_ids": ["a1b2c3d4-e5f6-7890-abcd-ef1234567890"]
}

response = requests.post(url, headers=headers, json=data)
print(response.json())
```
