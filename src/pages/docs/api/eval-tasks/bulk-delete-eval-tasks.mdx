# Bulk Delete Eval Tasks

Soft deletes multiple eval tasks and their associated logger and eval logger records in a single request.

<ApiPlayground
  method="POST"
  endpoint="/tracer/eval-task/mark_eval_tasks_deleted/"
  baseUrl="https://api.futureagi.com"
  parameters={[]}
  requestBody={{
    eval_task_ids: ["a1b2c3d4-e5f6-7890-abcd-ef1234567890"]
  }}
/>

## Authentication

All requests require authentication via API keys:

| Header | Description |
|--------|-------------|
| `X-Api-Key` | Your API key |
| `X-Secret-Key` | Your secret key |

## Request Body

<ParamField body="eval_task_ids" type="array" required>
  List of eval task UUIDs to mark as deleted.
</ParamField>

<Note>
  Running tasks cannot be deleted — they must be paused first. The request will fail if any task in the list is currently running.
</Note>

## Response

<ResponseField name="status" type="boolean">Indicates success.</ResponseField>
<ResponseField name="result" type="string">Confirmation message.</ResponseField>

### Responses

| Status | Description |
|--------|-------------|
| `200` | Eval tasks deleted successfully. |
| `400` | One or more tasks are running (must pause first) or task IDs do not exist. |
| `401` | Unauthorized. Authentication credentials were not provided or are invalid. |

## Code Examples

<CodeGroup>
```python Python
import requests

url = "https://api.futureagi.com/tracer/eval-task/mark_eval_tasks_deleted/"
headers = {
    "X-Api-Key": "YOUR_API_KEY",
    "X-Secret-Key": "YOUR_SECRET_KEY",
    "Content-Type": "application/json",
}

payload = {
    "eval_task_ids": ["a1b2c3d4-e5f6-7890-abcd-ef1234567890"],
}

response = requests.post(url, json=payload, headers=headers)
print(response.json())
```
