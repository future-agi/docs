# Create Eval Task

Creates a new eval task to run evaluations on trace spans within a project.

<ApiPlayground
  method="POST"
  endpoint="/tracer/eval-task/"
  baseUrl="https://api.futureagi.com"
  parameters={[]}
  requestBody={{"project": "your-project-uuid", "name": "my-eval-task", "evals": ["eval-config-uuid"], "sampling_rate": 100.0, "run_type": "continuous"}}
/>

## Authentication

All requests require authentication via API keys:

| Header | Description |
|--------|-------------|
| `X-Api-Key` | Your API key |
| `X-Secret-Key` | Your secret key |

## Request Body

- **project** (required): string (UUID)
  The project ID to associate this eval task with. Must belong to the user's organization.
- **name** (required): string
  Name of the eval task. Min length: 1, max length: 255 characters.
- **evals** (required): array of string (UUID)
  List of custom eval config IDs to run. Must contain at least one valid, non-deleted config.
- **sampling_rate** (required): number
  Percentage of spans to evaluate. Min: 1.0, max: 100.0.
- **run_type** (required): string
  Execution mode. Must be `"continuous"` or `"historical"`.
- **spans_limit**: integer
  Maximum number of spans to evaluate. Required when `run_type` is `"historical"`. Min: 1, max: 1,000,000. Ignored for `"continuous"` run type.
- **filters**: object
  Optional query filters to scope the spans evaluated.

### Example

```json
{
  "project": "a1b2c3d4-e5f6-7890-abcd-ef1234567890",
  "name": "my-eval-task",
  "evals": ["d4e5f6a7-b8c9-0123-def0-123456789abc"],
  "sampling_rate": 100.0,
  "run_type": "continuous",
  "filters": {}
}
```

## Responses

### 200

Eval task created successfully.

- **status**: boolean
- **result**: object
  Contains the `id` of the newly created eval task.

### 400

Bad Request. Validation failed. Possible reasons:
- Missing required fields (`project`, `name`, `evals`, `sampling_rate`, `run_type`).
- `evals` list is empty.
- `sampling_rate` outside 1.0–100.0 range.
- `spans_limit` missing for `"historical"` run type.
- `name` exceeds 255 characters.

### 401

Unauthorized. Authentication credentials were not provided or are invalid.

### 500

Internal Server Error. An unexpected error occurred.
