# Create Eval Task

Creates a new eval task to run evaluations on trace spans within a project.

<ApiPlayground
  method="POST"
  endpoint="/tracer/eval-task/"
  baseUrl="https://api.futureagi.com"
  parameters={[]}
  requestBody={{
    project: "a1b2c3d4-e5f6-7890-abcd-ef1234567890",
    name: "my-eval-task",
    evals: ["d4e5f6a7-b8c9-0123-def0-123456789abc"],
    sampling_rate: 100.0,
    run_type: "continuous"
  }}
/>

## Authentication

All requests require authentication via API keys:

| Header | Description |
|--------|-------------|
| `X-Api-Key` | Your API key |
| `X-Secret-Key` | Your secret key |

## Request Body

<ParamField body="project" type="string" required>
  UUID of the project to associate this eval task with. Must belong to the user's organization.
</ParamField>

<ParamField body="name" type="string" required>
  Name of the eval task. Must be between 1 and 255 characters.
</ParamField>

<ParamField body="evals" type="array" required>
  List of custom eval config UUIDs to run. Must contain at least one valid, non-deleted config.
</ParamField>

<ParamField body="sampling_rate" type="number" required>
  Percentage of spans to evaluate. Must be between `1.0` and `100.0`.
</ParamField>

<ParamField body="run_type" type="string" required>
  Execution mode. Must be `"continuous"` or `"historical"`.
</ParamField>

<ParamField body="spans_limit" type="integer">
  Maximum number of spans to evaluate. Required when `run_type` is `"historical"`. Must be between `1` and `1000000`. Ignored for `"continuous"` run type.
</ParamField>

<ParamField body="filters" type="object">
  Optional query filters to scope which spans are evaluated.
</ParamField>

## Response

<ResponseField name="id" type="string">UUID of the newly created eval task.</ResponseField>

### Responses

| Status | Description |
|--------|-------------|
| `200` | Eval task created successfully. |
| `400` | Validation failed — missing required fields, empty `evals`, `sampling_rate` out of range, or missing `spans_limit` for historical run type. |
| `401` | Unauthorized. Authentication credentials were not provided or are invalid. |
| `500` | Internal Server Error. |
