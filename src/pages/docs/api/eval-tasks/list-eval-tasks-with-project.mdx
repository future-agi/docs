# List Eval Tasks with Project Name

Retrieves a paginated list of eval tasks including their associated project names. Supports filtering and sorting.

## Authentication

All requests require authentication via API keys:

| Header | Description |
|--------|-------------|
| `X-Api-Key` | Your API key |
| `X-Secret-Key` | Your secret key |

## Parameters

### Query Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `page_number` | integer | No | Page number for pagination. Default: `0`. Also accepts `pageNumber`. |
| `page_size` | integer | No | Number of results per page. Default: `10`. Also accepts `pageSize`. |
| `filters` | string (JSON) | No | JSON-encoded filter criteria. |
| `sort_params` | string (JSON) | No | JSON-encoded sort criteria. Also accepts `sortParams`. |

## Responses

### 200

Successfully retrieved the list of eval tasks with project names.

- **status**: boolean
- **result**: object
