---
title: "Unpause Eval Task"
description: "Resumes a previously paused eval task. The task's status is reset to pending, and evaluation processing restarts from the beginning."
---

<ApiSection title="Query parameters">
  <ParamField query="eval_task_id" type="string" required>
    The unique identifier (UUID) of the eval task to resume. The task must currently be in a `paused` state. Tasks in any other state (e.g., `running`, `pending`, `completed`) cannot be unpaused and will return a `400` error. Note that when unpaused, the task status resets to `pending` and evaluation processing restarts from the beginning.
  </ParamField>
</ApiSection>

<ApiSection title="Response" status={200} statusText="OK">

  </ResponseField>

    A human-readable confirmation message describing the outcome of the operation.
  </ResponseField>
</ApiSection>
