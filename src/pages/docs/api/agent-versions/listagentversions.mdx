# List agent versions

Retrieves a paginated list of all versions for a specific agent definition, including version status, score, and test metrics.

<ApiPlayground
  method="GET"
  endpoint="/simulate/agent-definitions/{agent_id}/versions/"
  baseUrl="https://api.futureagi.com"
  parameters={[{"name": "agent_id", "in": "path", "required": true, "description": "A UUID string identifying the agent definition.", "type": "string"}, {"name": "limit", "in": "query", "required": false, "description": "Number of items per page.", "type": "integer"}, {"name": "page", "in": "query", "required": false, "description": "Page number.", "type": "integer"}]}
/>

## Authentication

All requests require authentication via API keys:

| Header | Description |
|--------|-------------|
| `X-Api-Key` | Your API key |
| `X-Secret-Key` | Your secret key |

## Parameters

### Path Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `agent_id` | string (UUID) | Yes | The UUID of the agent definition. |

### Query Parameters

| Parameter | Type | Required | Default | Description |
|-----------|------|----------|---------|-------------|
| `limit` | integer | No | 10 | Number of items per page. |
| `page` | integer | No | 1 | Page number. |

## Responses

### 200

A paginated list of agent versions.

- **count**: integer
- **next**: string or null
- **previous**: string or null
- **results**: array of objects
  Each object contains:
  - **id**: string (UUID)
  - **version_number**: integer
  - **version_name**: string
    Example: `v1`, `v2`.
  - **version_name_display**: string
    Formatted display name.
  - **status**: string (one of: `draft`, `active`, `archived`, `deprecated`)
  - **status_display**: string
  - **score**: number or null
    Score between 0.0 and 10.0.
  - **test_count**: integer
  - **pass_rate**: number or null
    Percentage between 0 and 100.
  - **description**: string
  - **commit_message**: string
  - **is_active**: boolean
  - **is_latest**: boolean
  - **created_at**: string (datetime)

### 401

Unauthorized. Authentication credentials were not provided or are invalid.

### 404

Not Found. The agent definition with the specified ID could not be found.

### 500

Internal Server Error. An unexpected error occurred.

```bash
  -H "X-Secret-Key: YOUR_SECRET_KEY"
```
