# Get eval summary for agent version

Retrieves the evaluation summary for a specific agent version, including pass/fail/error counts and pass rates for each configured evaluation template.

<ApiPlayground
  method="GET"
  endpoint="/simulate/agent-definitions/{agent_id}/versions/{version_id}/eval-summary/"
  baseUrl="https://api.futureagi.com"
  parameters={[{"name": "agent_id", "in": "path", "required": true, "description": "A UUID string identifying the agent definition.", "type": "string"}, {"name": "version_id", "in": "path", "required": true, "description": "A UUID string identifying the agent version.", "type": "string"}]}
/>

## Authentication

All requests require authentication via API keys:

| Header | Description |
|--------|-------------|
| `X-Api-Key` | Your API key |
| `X-Secret-Key` | Your secret key |

## Parameters

### Path Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `agent_id` | string (UUID) | Yes | The UUID of the agent definition. |
| `version_id` | string (UUID) | Yes | The UUID of the agent version. |

## Responses

### 200

Evaluation summary returned successfully. Returns an array of evaluation template statistics.

Each object in the array contains:
- **eval_id**: string (UUID)
- **eval_name**: string
- **eval_type**: string
- **total**: integer
  Total number of evaluations run.
- **pass**: integer
  Number of passed evaluations.
- **fail**: integer
  Number of failed evaluations.
- **error**: integer
  Number of errored evaluations.
- **pass_rate**: number
  Pass rate as a percentage.

Returns an empty array `[]` if no evaluation configurations exist.

### 401

Unauthorized. Authentication credentials were not provided or are invalid.

### 404

Not Found. The agent definition or version could not be found.

### 500

Internal Server Error. An unexpected error occurred.
