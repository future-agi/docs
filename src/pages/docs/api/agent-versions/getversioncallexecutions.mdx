  <ResponseField name="results" type="array">
    An array of call execution objects. Each object includes:
    - `id` — The unique identifier of the call execution.
    - `service_provider_call_id` — The call ID from the external provider (e.g., VAPI call ID). `null` if not available.
    - `session_id` — The session identifier linking related calls. `null` if not applicable.
    - `status` — The call status (only `completed` calls are returned).
    - `duration` — The call duration in seconds. `null` if not recorded.
    - `start_time` — ISO 8601 timestamp of when the call started.
    - `transcript` — An array of transcript turn objects from the conversation.
    - `scenario` — The name of the scenario used for this call.
    - `overall_score` — The aggregate evaluation score for the call. `null` if not yet scored.
    - `eval_outputs` — An object containing the detailed evaluation results for each configured eval template.
    - `eval_metrics` — Aggregated evaluation metrics. `null` if not available.
    - `scenario_columns` — Additional scenario column data associated with this call.
    - `customer_name` — The simulated customer name used in the call. `null` if not set.
    - `call_summary` — An AI-generated summary of the call. `null` if not generated.
    - `ended_reason` — The reason the call ended (e.g., `"completed"`, `"timeout"`, `"error"`). `null` if not recorded.
    - `created_at` — ISO 8601 timestamp of when the call execution was created.
  </ResponseField>
