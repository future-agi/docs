# Get agent version details

Retrieves detailed information about a specific version of an agent definition, including the full configuration snapshot captured when the version was created.

<ApiPlayground
  method="GET"
  endpoint="/simulate/agent-definitions/{agent_id}/versions/{version_id}/"
  baseUrl="https://api.futureagi.com"
  parameters={[{"name": "agent_id", "in": "path", "required": true, "description": "A UUID string identifying the agent definition.", "type": "string"}, {"name": "version_id", "in": "path", "required": true, "description": "A UUID string identifying the agent version.", "type": "string"}]}
/>

## Authentication

All requests require authentication via API keys:

| Header | Description |
|--------|-------------|
| `X-Api-Key` | Your API key |
| `X-Secret-Key` | Your secret key |

## Parameters

### Path Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `agent_id` | string (UUID) | Yes | The UUID of the agent definition. |
| `version_id` | string (UUID) | Yes | The UUID of the agent version. |

## Responses

### 200

Agent version details returned successfully.

- **id**: string (UUID)
- **version_number**: integer
- **version_name**: string
- **version_name_display**: string
- **status**: string (one of: `draft`, `active`, `archived`, `deprecated`)
- **status_display**: string
- **score**: number or null
  Score between 0.0 and 10.0.
- **test_count**: integer
- **pass_rate**: number or null
  Percentage between 0 and 100.
- **description**: string
- **commit_message**: string
- **release_notes**: string or null
- **agent_definition**: string (UUID)
- **organization**: string (UUID)
- **configuration_snapshot**: object
  Snapshot of the agent configuration at version creation time:
  - **agent_name**: string
  - **agent_type**: string or null
  - **inbound**: boolean
  - **languages**: array of string
  - **language**: string
  - **provider**: string or null
  - **contact_number**: string or null
  - **assistant_id**: string or null
  - **api_key**: string or null
  - **authentication_method**: string
  - **observability_enabled**: boolean
  - **description**: string or null
  - **knowledge_base**: string or null
  - **commit_message**: string
  - **model**: string or null
  - **model_details**: object or null
- **is_active**: boolean
- **is_latest**: boolean
- **created_at**: string (datetime)
- **updated_at**: string (datetime)

### 401

Unauthorized. Authentication credentials were not provided or are invalid.

### 404

Not Found. The agent definition or version could not be found.

### 500

Internal Server Error. An unexpected error occurred.
