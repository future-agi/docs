# Edit Run Prompt Column

Update the configuration of an existing run prompt column. This clears all existing cell values and re-executes the prompt with the new configuration for all rows.

<ApiPlayground
  method="POST"
  endpoint="/model-hub/develops/edit_run_prompt_column/"
  baseUrl="https://api.futureagi.com"
  parameters={[]}
  requestBody={{
    dataset_id: "a1b2c3d4-e5f6-7890-abcd-ef1234567890",
    column_id: "b2c3d4e5-f6a7-8901-bcde-f12345678901",
    name: "gpt_response_v2",
    config: {
      model: "gpt-4o-mini",
      messages: [
        { role: "user", content: "Answer: {{question}}" }
      ],
      temperature: 0.5
    }
  }}
/>

## Authentication

All requests require authentication via API keys:

| Header | Description |
|--------|-------------|
| `X-Api-Key` | Your API key |
| `X-Secret-Key` | Your secret key |

## Request Body

<ParamField body="dataset_id" type="string (UUID)" required>
  The UUID of the dataset.
</ParamField>

<ParamField body="column_id" type="string (UUID)" required>
  The UUID of the run prompt column to edit.
</ParamField>

<ParamField body="name" type="string">
  New name for the column. If not provided, the existing name is kept.
</ParamField>

<ParamField body="config" type="object">
  Updated prompt configuration. See [Add Run Prompt Column](/docs/api/datasets/run-prompt/add-run-prompt-column) for the full config schema.
</ParamField>

### Example

```json
{
  "dataset_id": "a1b2c3d4-e5f6-7890-abcd-ef1234567890",
  "column_id": "b2c3d4e5-f6a7-8901-bcde-f12345678901",
  "name": "gpt_response_v2",
  "config": {
    "model": "gpt-4o-mini",
    "messages": [
      {
        "role": "user",
        "content": "Answer the following question:\n\n{{question}}"
      }
    ],
    "temperature": 0.5,
    "max_tokens": 512
  }
}
```

## Response

<ResponseField name="message" type="string">Confirmation message indicating the column was updated.</ResponseField>

### Example Response

```json
{
  "message": "Run prompt column updated successfully"
}
```

## Responses

### 200

Column updated and prompt re-execution started. All existing cell values are cleared and regenerated asynchronously.

### 400

Bad request. Possible reasons:

- **Invalid column** — The specified column is not a RUN_PROMPT column.
- **Invalid config** — The prompt configuration is invalid.

### 401

Authentication credentials were not provided or are invalid.

### 404

Dataset or column not found, or does not belong to your organization.

### 500

Internal server error.
