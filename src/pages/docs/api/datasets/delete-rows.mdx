<ApiSection title="Request body">
  <ParamField body="row_ids" type="array" optional>
    An array of row UUIDs to delete from the dataset. Each entry must be a valid UUID string corresponding to an existing row in the specified dataset. This field is required when `selected_all_rows` is set to `false` or is omitted. The rows will be permanently removed and cannot be recovered.
  </ParamField>
  <ParamField body="selected_all_rows" type="boolean" optional>
    A boolean flag that, when set to `true`, instructs the API to delete every row in the dataset, effectively clearing all data while preserving the dataset structure and columns. Defaults to `false`. When set to `true`, the `row_ids` field is ignored.
  </ParamField>
</ApiSection>
