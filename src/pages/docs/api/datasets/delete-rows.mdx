# Delete Rows

Delete specific rows from a dataset by their IDs, or delete all rows.

<ApiPlayground
  method="DELETE"
  endpoint="/model-hub/develops/{id}/delete_row/"
  baseUrl="https://api.futureagi.com"
  parameters={[{"name": "id", "in": "path", "required": true, "description": "UUID of the dataset.", "type": "string"}]}
  requestBody={{
    row_ids: ["row-uuid-1", "row-uuid-2"],
    selected_all_rows: false
  }}
/>

## Authentication

All requests require authentication via API keys:

| Header | Description |
|--------|-------------|
| `X-Api-Key` | Your API key |
| `X-Secret-Key` | Your secret key |

## Parameters

### Path Parameters

<ParamField path="id" type="string" required>
  UUID of the dataset containing the rows to delete.
</ParamField>

## Request Body

<ParamField body="row_ids" type="array">
  Array of row UUIDs to delete. Required if `selected_all_rows` is `false`.
</ParamField>

<ParamField body="selected_all_rows" type="boolean">
  Set to `true` to delete all rows in the dataset. When `true`, `row_ids` is ignored. Default: `false`.
</ParamField>

### Example

```json
{
  "row_ids": [
    "row-uuid-1",
    "row-uuid-2"
  ],
  "selected_all_rows": false
}
```

## Response

Returns confirmation of deleted rows.

<ResponseField name="data" type="string">Confirmation message.</ResponseField>
<ResponseField name="status" type="string">Status indicator. Example: `success`.</ResponseField>

### Example Response

```json
{
  "data": "Row deleted successfully",
  "status": "success"
}
```

## Responses

### 200

Rows deleted successfully.

### 400

Bad request. Possible reasons:

- **Missing rows** - Either `row_ids` or `selected_all_rows` must be provided.
- **No matching rows** - The provided row IDs do not exist in the dataset.

### 401

Authentication credentials were not provided or are invalid.

### 404

Dataset not found.

### 500

Internal server error.

## Code Examples

<CodeGroup>
```python Python
import requests

url = "https://api.futureagi.com/model-hub/develops/a1b2c3d4-e5f6-7890-abcd-ef1234567890/delete_row/"
headers = {
    "X-Api-Key": "YOUR_API_KEY",
    "X-Secret-Key": "YOUR_SECRET_KEY",
    "Content-Type": "application/json"
}
data = {
    "row_ids": ["row-uuid-1", "row-uuid-2"],
    "selected_all_rows": False
}

response = requests.delete(url, headers=headers, json=data)
print(response.json())
```
