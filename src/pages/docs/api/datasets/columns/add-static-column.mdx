<ApiSection title="Path parameters">
  <ParamField path="dataset_id" type="string" required>
    The universally unique identifier (UUID) of the dataset to which you want to add a static column. This ID is returned when you create a dataset and can be found in the dataset listing or dataset detail endpoints.
  </ParamField>
</ApiSection>

<ApiSection title="Request body">
  <ParamField body="new_column_name" type="string" required>
    The human-readable name to assign to the new static column. This name is used to identify the column throughout the Future AGI platform, including in the dataset grid, evaluation configurations, and prompt template variable mappings. Must be unique within the dataset.
  </ParamField>

  <ParamField body="column_type" type="string" required>
    The data type for the new column, which determines what kind of values can be stored in it and how the platform renders and validates cell contents. Valid values: `text`, `number`, `boolean`, `json`, `image`, `audio`, `pdf`.
  </ParamField>

  <ParamField body="source" type="string" optional>
    An optional source type annotation for the column. When specified, this indicates the origin or intended purpose of the column data, which can be used by the platform for downstream processing and filtering.
  </ParamField>
</ApiSection>

<ApiSection title="Response" status={200} statusText="OK">
  <ResponseField name="message" type="string">
    A human-readable confirmation message indicating that the column was created successfully. Empty cells are automatically initialized for all existing rows in the dataset. Typically returns `Column added successfully`.
  </ResponseField>
</ApiSection>
