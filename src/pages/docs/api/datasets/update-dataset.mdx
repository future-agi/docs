# Update Dataset

Update the properties of an existing dataset. Currently supports renaming a dataset.

<ApiPlayground
  method="POST"
  endpoint="/model-hub/dataset/{id}/"
  baseUrl="https://api.futureagi.com"
  parameters={[{"name": "id", "in": "path", "required": true, "description": "UUID of the dataset to update.", "type": "string"}]}
  requestBody={{
    name: "Updated Dataset Name"
  }}
/>

## Authentication

All requests require authentication via API keys:

| Header | Description |
|--------|-------------|
| `X-Api-Key` | Your API key |
| `X-Secret-Key` | Your secret key |

## Parameters

### Path Parameters

<ParamField path="id" type="string" required>
  UUID of the dataset to update.
</ParamField>

## Request Body

<ParamField body="name" type="string" required>
  New name for the dataset. Must be unique within your organization.
</ParamField>

### Example

```json
{
  "name": "Updated Dataset Name"
}
```

## Response

Returns the updated dataset details.

<ResponseField name="dataset_id" type="string">UUID of the updated dataset.</ResponseField>
<ResponseField name="name" type="string">Updated name of the dataset.</ResponseField>

### Example Response

```json
{
  "dataset_id": "a1b2c3d4-e5f6-7890-abcd-ef1234567890",
  "name": "Updated Dataset Name"
}
```

## Responses

### 200

Dataset updated successfully.

### 400

Bad request. Possible reasons:

- **Duplicate name** - A dataset with this name already exists in your organization.
- **Missing name** - The `name` field is required.

### 401

Authentication credentials were not provided or are invalid.

### 404

Dataset not found. The provided ID does not match any dataset in your organization.

### 500

Internal server error.

```python Python
import requests

url = "https://api.futureagi.com/model-hub/dataset/a1b2c3d4-e5f6-7890-abcd-ef1234567890/"

    "X-Api-Key": "YOUR_API_KEY",
    "X-Secret-Key": "YOUR_SECRET_KEY",
    "Content-Type": "application/json"
}
data = {
    "name": "Updated Dataset Name"
}

response = requests.post(url, headers=headers, json=data)
print(response.json())
```
