# Clone Dataset

Create a full copy of an existing dataset including all columns (user-created) and rows.

<ApiPlayground
  method="POST"
  endpoint="/model-hub/develops/clone-dataset/{id}/"
  baseUrl="https://api.futureagi.com"
  parameters={[{"name": "id", "in": "path", "required": true, "description": "UUID of the dataset to clone.", "type": "string"}]}
  requestBody={{
    new_dataset_name: "Copy of My Dataset"
  }}
/>

## Authentication

All requests require authentication via API keys:

| Header | Description |
|--------|-------------|
| `X-Api-Key` | Your API key |
| `X-Secret-Key` | Your secret key |

## Parameters

### Path Parameters

<ParamField path="id" type="string" required>
  UUID of the source dataset to clone.
</ParamField>

## Request Body

<ParamField body="new_dataset_name" type="string">
  Name for the cloned dataset. If not provided, defaults to `Copy of {source_dataset_name}`. Must be unique within your organization.
</ParamField>

### Example

```json
{
  "new_dataset_name": "Copy of My Dataset"
}
```

## Response

Returns the cloned dataset details.

<ResponseField name="message" type="string">Confirmation message.</ResponseField>
<ResponseField name="dataset_id" type="string">UUID of the newly cloned dataset.</ResponseField>
<ResponseField name="dataset_name" type="string">Name of the cloned dataset.</ResponseField>

### Example Response

```json
{
  "message": "Dataset cloned successfully",
  "dataset_id": "b2c3d4e5-f6a7-8901-bcde-f12345678901",
  "dataset_name": "Copy of My Dataset"
}
```

## Responses

### 200

Dataset cloned successfully.

### 400

Bad request. Possible reasons:

- **Duplicate name** - A dataset with this name already exists in your organization.

### 401

Authentication credentials were not provided or are invalid.

### 404

Source dataset not found.

### 429

Resource limit reached. Your organization has exceeded the dataset creation or row addition quota.

### 500

Internal server error.

```python Python
import requests

    "X-Api-Key": "YOUR_API_KEY",
    "X-Secret-Key": "YOUR_SECRET_KEY",
    "Content-Type": "application/json"
}

    "new_dataset_name": "Copy of My Dataset"

response = requests.post(url, headers=headers, json=data)
print(response.json())
```
