# Edit Evaluation Group Members

Adds or removes evaluation templates from an evaluation group.

<ApiPlayground
  method="POST"
  endpoint="/model-hub/eval-groups/edit-eval-list/"
  baseUrl="https://api.futureagi.com"
  parameters={[]}
  requestBody={{"eval_group_id": "your-eval_group_id", "added_template_ids": [], "deleted_template_ids": []}}
/>


## Authentication

This endpoint requires authentication. Include your API key in the Authorization header:

```bash
Authorization: Bearer YOUR_API_KEY
```

## Request Body

- **eval_group_id** (required): string
  The UUID of the evaluation group to modify.
- **added_template_ids**: array of string
  A list of evaluation template UUIDs to add to the group.
- **deleted_template_ids**: array of string
  A list of evaluation template UUIDs to remove from the group.

### Example

```json
{
  "eval_group_id": "your-eval_group_id",
  "added_template_ids": [],
  "deleted_template_ids": []
}
```

## Responses

### 200

Evaluation group updated successfully.

### 400

Bad Request. The request body is invalid or the group was not found.



### 401

Unauthorized. Authentication credentials were not provided or are invalid.

### 500

Internal Server Error. An unexpected error occurred.



## Code Examples

### cURL

```bash
curl -X POST "https://api.futureagi.com/model-hub/eval-groups/edit-eval-list/" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"eval_group_id": "your-eval_group_id", "added_template_ids": [], "deleted_template_ids": []}'
```

### Python

```python
import requests

url = "https://api.futureagi.com/model-hub/eval-groups/edit-eval-list/"
headers = {
    "Authorization": "Bearer YOUR_API_KEY",
    "Content-Type": "application/json"
}
data = {
    "eval_group_id": "your-eval_group_id",
    "added_template_ids": [],
    "deleted_template_ids": []
}

response = requests.post(url, headers=headers, json=data)
print(response.json())
```
