<TLDR>
Future AGI's evaluation suite lets you measure LLM output quality with 70+ built-in metrics covering relevance, correctness, safety, and RAG-specific measurements. Run evaluations locally or in the cloud with a single API call.
</TLDR>

## Why Evaluate?

Building reliable AI applications requires continuous measurement. Without evaluation:

- **Quality regression** goes unnoticed until users complain
- **Safety issues** slip through during rapid iteration
- **RAG performance** degrades as knowledge bases grow
- **Cost optimization** becomes guesswork

Future AGI's evaluation framework provides objective metrics to catch issues before production.

## Built-in Metrics

Our evaluation suite includes 70+ metrics organized by use case:

### Quality Metrics

| Metric | Description |
|--------|-------------|
| `relevance` | How well the output addresses the input query |
| `correctness` | Factual accuracy of the response |
| `coherence` | Logical flow and internal consistency |
| `completeness` | Whether all aspects of the query are addressed |
| `conciseness` | Information density without unnecessary verbosity |

### Safety Metrics

| Metric | Description |
|--------|-------------|
| `toxicity` | Detection of harmful or offensive content |
| `bias` | Identification of unfair stereotypes or prejudice |
| `pii_detection` | Personal identifiable information exposure |
| `jailbreak_detection` | Prompt injection attempt identification |

### RAG Metrics

| Metric | Description |
|--------|-------------|
| `context_relevance` | Retrieved context quality for the query |
| `faithfulness` | Output grounded in provided context |
| `answer_relevance` | Response relevance to original question |
| `context_recall` | Coverage of relevant information |

## Quick Example

```python
from futureagi import FutureAGI

client = FutureAGI()

# Simple evaluation
result = client.evaluate(
    input="What causes rain?",
    output="Rain is caused by water vapor condensing in clouds.",
    metrics=["relevance", "correctness", "coherence"]
)

print(result.scores)
# {'relevance': 0.92, 'correctness': 0.95, 'coherence': 0.98}
```

## RAG Evaluation

For retrieval-augmented generation, include context:

```python
result = client.evaluate(
    input="What is our refund policy?",
    output="Our refund policy allows returns within 30 days.",
    context=["Returns are accepted within 30 days of purchase..."],
    metrics=["faithfulness", "context_relevance", "answer_relevance"]
)
```

<Callout type="tip">
Use `client.evaluate_batch()` to run evaluations on entire datasets efficiently.
</Callout>
