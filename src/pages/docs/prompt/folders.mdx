## What it is

**Prompt folders** let you organize and categorize your prompt templates by grouping related prompts together. This structure helps teams manage large prompt libraries, keep use cases and projects easy to find, and maintain a clear layout as the library grows. You create folders in the Prompt Workbench (e.g. via the **New folder** button), then move or create prompts inside them.

---

## Use cases

- **Group related prompts** — Organize by function (e.g. summarization, support), team, or project so prompts are easy to find.
- **Improve navigation** — Quickly locate specific prompt templates instead of scrolling a flat list.
- **Maintain structure** — Keep the prompt library organized as you add more templates.
- **Team collaboration** — Share a consistent folder structure so everyone knows where to find or add prompts.

---

## How to

### Creating folders

In the Prompts section, use the **New folder** button to create a folder. You can then:

- **Group related prompts** — Put prompts that belong together (e.g. by functionality, team, or project) in the same folder.
- **Improve navigation** — Make it easier to find specific prompt templates by browsing folders instead of a long list.
- **Maintain structure** — Keep your prompt library organized as it grows.
- **Support team collaboration** — Use a shared folder structure so the whole team can find and manage prompts in the same way.
