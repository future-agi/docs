---
title: Content Analysis
description: 'Evaluate your LLM models on their ability to generate text based on content criteria.'
---

Content analysis refers to the ability to evaluate and validate text output from LLM models based on various criteria. This includes checking for specific content and text patterns.
#### Required Arguments

- `text`: The text to analyze

| Validation Type | Description |
|----------------|-------------|
| `Regex` | Validate text against a regular expression pattern |
| `ContainsAny` | Check if text contains any of the specified strings |
| `ContainsAll` | Check if text contains all of the specified strings |
| `Contains` | Check if text contains a specific string |
| `ContainsNone` | Check if text contains none of the specified strings |
| `Equals` | Check if text exactly matches a string |
| `StartsWith` | Check if text starts with a string |
| `EndsWith` | Check if text ends with a string |


```
from fi.evals import Regex, EvalClient
from fi.evals.testcases import TestCase

client = EvalClient()
test_case = TestCase(
    text="Hello, world!"
)

result = client.evaluate(test_case, Regex, config={"pattern": "^Hello, world!$"})
```
