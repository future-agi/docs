title: Content Analysis
description: 'Evaluate your LLM models on their ability to generate text based on content criteria.'
---

Content analysis refers to the ability to evaluate and validate text output from LLM models based on various criteria. This includes checking for specific content and text patterns.

| Evaluation Type | Description | Required Configuration |
|----------------|-------------|--------------------|
| `Regex` | Validate text against a regular expression pattern | 1. `pattern`: The regular expression pattern to validate against |
| `ContainsAny` | Check if text contains any of the specified strings | 1. `keywords`: List of strings to check for, <br /> 2. `case_sensitive`: Whether comparison is case sensitive (default: true) |
| `ContainsAll` | Check if text contains all of the specified strings | 1. `keywords`: List of strings to check for, <br /> 2. `case_sensitive`: Whether comparison is case sensitive (default: true) |
| `Contains` | Check if text contains a specific string | 1. `keyword`: String to check for, <br /> 2. `case_sensitive`: Whether comparison is case sensitive (default: true) |
| `ContainsNone` | Check if text contains none of the specified strings | 1. `keywords`: List of strings to check for, <br /> 2. `case_sensitive`: Whether comparison is case sensitive (default: true) |
| `Equals` | Check if text exactly matches a string | 1. `case_sensitive`: Whether comparison is case sensitive (default: true) |
| `StartsWith` | Check if text starts with a string | 1. `substring`: String to check for at start, <br /> 2. `case_sensitive`: Whether comparison is case sensitive (default: true) |
| `EndsWith` | Check if text ends with a string | 1. `substring`: String to check for at end, <br /> 2. `case_sensitive`: Whether comparison is case sensitive (default: true) |


```python
from fi.evals import Regex, EvalClient
from fi.evals.testcases import TestCase

evaluator = EvalClient(fi_api_key="your_api_key", fi_secret_key="your_secret_key")
regex_test_case = TestCase(text="Narendra Modi")
regex_template = Regex(config={"pattern": "Narendra Modi"})

result = evaluator.evaluate(regex_template, regex_test_case)
print(result)
```
