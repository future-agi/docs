---
title: Conversation Resolution
description: 'Evaluate your LLM models on their ability to resolve a conversation.'
---


Conversation resolution refers to the ability of an LLM model to effectively and accurately resolve a conversation. It ensures that the exchange of information or ideas between participants aligns with the context and builds meaningfully upon prior interactions. In the context of natural language processing and conversational AI, resolution is a critical metric to evaluate how well an AI or chatbot can maintain a contextually relevant and natural conversation.

### Configuration
The evaluation requires the following configuration:

| Parameter | Description |
|-----------|-------------|
| `model`   | Optional: The model to be used for evaluation |

```python
from fi.evals import ConversationResolution

resolution = ConversationResolution(config = {"model": "gpt-4o-mini"})
```

### Test Case Setup
The evaluation requires a list of messages between the user and assistant:

```python
from fi.test_cases import ConversationalTestCase

test_case = ConversationalTestCase(
    messages=[
        {"role": "user", "content": "Hello, how are you?"},
        {"role": "assistant", "content": "I am fine, thank you."}
    ]
)
```

### Client Setup
Initialize the evaluation client with your API credentials:

```python
from fi.evals import EvalClient

evaluator = EvalClient(
    fi_api_key="your_api_key", 
    fi_secret_key="your_secret_key"
)
```

## Complete Example
```python
from fi.evals import ConversationResolution, EvalClient
from fi.test_cases import ConversationalTestCase

resolution = ConversationResolution()

test_case = ConversationalTestCase(
    messages=[
        {"role": "user", "content": "Hello, how are you?"},
        {"role": "assistant", "content": "I am fine, thank you."}
    ]
)

evaluator = EvalClient(fi_api_key="your_api_key", fi_secret_key="your_secret_key")
result = evaluator.evaluate(resolution, test_case)
print(result)
```
