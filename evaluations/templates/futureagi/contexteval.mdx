---
title: Context Evaluation
description: 'Scores how well context generates output based on specified criteria'
---


Context evaluation assesses how effectively a given context generates the desired output based on specified evaluation criteria. This evaluation type is particularly useful when you need to validate whether the provided context contains sufficient and relevant information to produce accurate and appropriate responses.

The evaluation analyzes various aspects of the context-output relationship, including:
- Relevance: How well the context aligns with the required output
- Completeness: Whether the context contains all necessary information
- Accuracy: The precision of information contained in the context
- Coherence: How logically structured and connected the context is

### Configuration
The evaluation requires the following configuration:

| Parameter | Description |
|-----------|-------------|
| `model` | Optional: The model to be used for evaluation |
| `criteria` | Optional: Custom evaluation criteria for context assessment |
