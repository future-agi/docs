---
title: Deterministic Evaluation
description: 'Evaluate your LLM models on their ability to generate deterministic outputs.'
---

Deterministic evaluation assesses whether a model's responses are consistent and predictable when given the same input multiple times. This evaluation is crucial for applications that require reliable and reproducible outputs.

### Configuration
The evaluation accepts the following configuration parameters:

| Parameter | Description | Required | Default |
|-----------|-------------|-----------|---------|
| `multi_choice` | Whether to allow multiple choices in output | No | `false` |
| `choices` | List of valid choices/outputs | No | `[]` |
| `rule_prompt` | Custom prompt for evaluation rules | No | `""` |
| `input` | Input strings to test | No | `[]` |

```python
from fi.evals import DeterministicEvaluation

# Initialize the deterministic evaluator
deterministic_eval = DeterministicEvaluation(config={
    "multi_choice": False,
    "choices": ["Yes", "No"],
    "rule_prompt": "Evaluate if the response is consistently the same for identical inputs",
    "input": ["test_input_1", "test_input_2"]
})  
```

### Test Case Setup
The evaluation requires test cases with inputs and outputs:

```python
from fi.testcases import TestCase

test_case = TestCase(
    input="What is 2+2?",
    output="4"
)
```

### Client Setup
Initialize the evaluation client with your API credentials:

```python
from fi.evals import EvalClient

evaluator = EvalClient(fi_api_key="your_api_key", fi_secret_key="your_secret_key")
```

### Complete Example

```python
from fi.evals import DeterministicEvaluation, EvalClient
from fi.testcases import TestCase

# Initialize the deterministic evaluator
deterministic_eval = DeterministicEvaluation(config={
    "multi_choice": False,
    "choices": ["4", "four"],
    "rule_prompt": "Evaluate if the response consistently provides the same answer",
    "input": ["What is 2+2?"]
})

# Create a test case
test_case = TestCase(input = "What is 2+2?", output = "4")

# Initialize the evaluation client
evaluator = EvalClient(
    fi_api_key="your_api_key",
    fi_secret_key="your_secret_key"
)

# Run the evaluation
result = evaluator.evaluate(deterministic_eval, test_case)
print(result) # Will return Pass if output is deterministic
```
