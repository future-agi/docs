---
title: Image Instruction Evaluation
description: 'Evaluate your LLM models on their ability to generate images based on custom criteria.'
---

Image instruction evaluation assesses how well an instruction aligns with and describes the corresponding image. This evaluation type is particularly useful for validating image generation models and ensuring that generated images match the given instructions or prompts.

### Required Configuration

| Parameter | Description | Required |
|-----------|-------------|-----------|
| `input` | The instruction to be evaluated | Yes |
| `image_url` | The URL of the image | Yes |
| `criteria` | The evaluation criteria | Yes |

### Example

```python
from fi.evals import ImageInstruction, EvalClient
from fi.testcases import ImageTestCase

# Initialize the evaluation client
evaluator = EvalClient(
    fi_api_key="your_api_key",
    fi_secret_key="your_secret_key"
)

# Create a test case
test_case = ImageTestCase(
    input="Generate a realistic photo of a red apple on a wooden table",
    image_url="https://example.com/apple-image.jpg",
    criteria="""
    Evaluate the image based on:
    1. Accuracy of object representation (apple)
    2. Setting accuracy (wooden table)
    3. Color accuracy (red)
    4. Overall composition
    5. Image quality and realism
    
    Each criterion is scored 0-1, with final score being average.
    """
)

# Initialize the image instruction evaluator
image_eval = ImageInstruction()

# Run the evaluation
result = evaluator.evaluate(image_eval, test_case)
print(result)  # Will return a score between 0 and 1
```

### Use Cases

- **Image Generation Models**: Evaluate how well generated images match input prompts
- **Image Description Validation**: Verify accuracy of image descriptions
- **Visual AI Training**: Assess instruction-to-image alignment
- **Quality Control**: Ensure generated images meet specified criteria

### Best Practices

1. **Clear Instructions**: Provide specific, unambiguous instructions
2. **Detailed Criteria**: Define comprehensive evaluation criteria
3. **Objective Metrics**: Include measurable aspects in criteria
4. **Consistent Scoring**: Use standardized scoring methods
5. **Multiple Aspects**: Consider various image qualities (composition, color, accuracy)

The evaluation returns a score between 0 and 1, where:
- 1.0: Perfect alignment between instruction and image
- 0.0: Complete mismatch between instruction and image
