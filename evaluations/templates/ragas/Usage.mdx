RAGAS provides a comprehensive suite of evaluation metrics specifically designed for RAG (Retrieval-Augmented Generation) systems. These metrics help assess various aspects of your RAG pipeline's performance.

## Available Metrics

| Metric | Description | Required Parameters |
|--------|-------------|-------------------|
| `RagasAnswerCorrectness` | Evaluates if the answer is factually correct | `expected_response`<br/> `response`<br/> `query` |
| `RagasAnswerRelevancy` | Measures how relevant the answer is to the query | `response`<br/> `context`<br/> `query` |
| `RagasAnswerSemanticSimilarity` | Compares semantic similarity between response and expected response | `response`<br/> `expected_response` |
| `RagasCoherence` | Evaluates response coherence and readability | `response` |
| `RagasConciseness` | Measures how concise and focused the response is | `response` |
| `RagasContextPrecision` | Evaluates precision of retrieved context | `expected_response`<br/> `context`<br/> `query` |
| `RagasContextRecall` | Measures completeness of retrieved context | `expected_response`<br/> `context`<br/> `query` |
| `RagasContextRelevancy` | Assesses relevance of retrieved context to query | `context`<br/> `query` |
| `RagasFaithfulness` | Measures response's faithfulness to provided context | `response`<br/> `context`<br/> `query` |
| `RagasHarmfulness` | Detects harmful content in responses | `response` |
| `RagasMaliciousness` | Identifies malicious content in responses | `response` |


## Configuration

All RAGAS metrics accept an optional configuration parameter:

| Parameter | Description | Required |
|-----------|-------------|----------|
| `model` | The LLM model to use for evaluation | Yes |



## Example Usage

```python
from fi.evals import RagasAnswerCorrectness, EvalClient
from fi.testcases import LLMTestCase

# Initialize the evaluation client
evaluator = EvalClient(
    fi_api_key="your_api_key",
    fi_secret_key="your_secret_key"
)

# Create a test case
test_case = LLMTestCase(
    query="What is the capital of France?",
    response="The capital of France is Paris.",
    expected_response="Paris is the capital city of France.",
    context=["Paris is the capital and largest city of France."]
)

# Initialize the RAGAS evaluator
ragas_eval = RagasAnswerCorrectness(config={
    "model": "gpt-4o-mini"  # Optional, defaults to system default
})

# Run the evaluation
result = evaluator.evaluate(ragas_eval, test_case)
print(result)  # Will return a score between 0 and 1
```
