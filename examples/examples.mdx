## Setup

First, import the required dependencies and initialize the client:

```python
from fi.client import Client
from fi.utils.types import ModelTypes, Environments

logger = Client(api_key="fi_api_key", secret_key="fi_secret_key",uri="fi_base_url")
```

## Example 1: Basic Chat History Logging

Log a single chat history event:

```python
logger.log(
    model_id="model_1",
    model_type=ModelTypes.GENERATIVE_LLM,
    environment=Environments.PRODUCTION,
    conversation={
        "chat_history": [
            {
                "role": "user",
                "content": "Hello!",
                "context": [["part1", "part2"]]
            }
        ]
    }
)
```

## Example 2: Chat History with Conversation ID

Log a chat history event with a conversation identifier:

```python
logger.log(
    model_id="model_1",
    model_type=ModelTypes.GENERATIVE_LLM,
    environment=Environments.PRODUCTION,
    conversation={
        "chat_history": [
            {
                "role": "user",
                "content": "Hello!",
                "context": [["part1", "part2"]]
            }
        ],
        "conversation_id": "conv_1"
    }
)
```

## Example 3: Advanced Chat Graph Logging

Log a more complex chat event using the graph format:

```python
logger.log(
    model_id="model_2",
    model_type=ModelTypes.GENERATIVE_LLM,
    environment=Environments.STAGING,
    conversation={
        "chat_graph": {
            "conversation_id": "conv_1",
            "nodes": [
                {
                    "message": {
                        "id": "msg_1",
                        "author": {"role": "user", "metadata": {}},
                        "content": {"content_type": "text", "parts": ["Hello"]},
                        "context": [["part1", "part2"]]
                    },
                    "node_id": "node_1",
                    "parent_id": "node_0",
                    "timestamp": 1625216400
                }
            ]
        }
    },
    tags={"project": "AI project"}
)
```
